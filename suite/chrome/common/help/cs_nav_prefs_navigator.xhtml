Préférences du navigateur - &navigator.label;

Cette section décrit comment utiliser le panneau principal des préférences du navigateur. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionner &preferencesCmd.label;.
  2. Sélectionnez la catégorie &navigator.label;.

Le panneau des préférences du navigateur vous permet de personnaliser certains aspects du navigateur.

&navRadio.label; :
Choisissez dans la liste déroulante l'élément dont vous voulez configurer l'ouverture :
  • &navStartPageMenu.label; ;
  • &newWinPageMenu.label; ;
  • &newTabPageMenu.label;.

Sélectionnez ensuite un de ces trois boutons pour déterminer ce que doit afficher l'élément lors de son ouverture :

&blankPageRadio.label; :
Le navigateur s'ouvre sans chargement automatique de page Web.
&homePageRadio.label; :
Le navigateur charge une page d'accueil (spécifiée ci-dessous) chaque fois qu'il est lancé.
&lastPageRadio.label; :
Le navigateur démarre en chargeant la dernière page visitée lors du dernier arrêt de &brandShortName;.
Navigateur par défaut :
Vous permet de définir &brandShortName; comme navigateur par défaut. Cette option n'est disponible que pour la version Microsfot® Windows de &brandShortName;.
&header2.label; :
Saisissez l'adresse de la page Web que vous voulez utiliser en page d'accueil, ou cliquez sur un des boutons suivants :
&useCurrent.label; :
Cliquez ici pour choisir la page Web en cours d'affichage sur le navigateur.
&useCurrentGroup.label; :
Si vous avez plus d'un onglet d'ouvert, ce bouton vous permet de définir ceux-ci comme pages d'accueil à lancer au démarrage de &brandShortName;. Après avoir cliqué sur ce bouton, vous verrez s'afficher le message Le groupe de pages d'accueil est fixé. Pour plus d'informations concernant l'ouverture d'onglets, voir la section Navigation par onglets.

Attention : Si vous éditez le champ texte après avoir appuyé sur le bouton &useCurrent.label;, vous perdrez le groupe d'onglets à afficher au démarrage.

&browseFile.label; :
Cliquez sur ce bouton pour rechercher sur votre disque dur, un fichier que vous voulez charger pour votre page d'accueil.
&useDefault.label; :
Cliquez sur ce bouton pour rétablir la page d'accueil par défaut de &brandShortName;.
&toolbarIntro.label; :
Sélectionnez une ou plusieurs des cases à cocher disponibles pour voir dans vos barres d'outils les boutons correspondants.

Les boutons &goButton.label; et &searchButton.label; apparaissent dans la barre d'outils du navigateur, près du coin en haut à droite de la fenêtre du navigateur. Pour plus d'informations concernant l'ajout de vos marque-pages dans cette barre d'outils, voir la section &navToolbarIntro.label;.

Préférences du navigateur - &history.label;

Cette section décrit comment utiliser le panneau des préférences de l'historique. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionner &preferencesCmd.label;.
  2. Dans la catégorie &navigator.label;, cliquez sur &history.label;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &navigator.label; pour développer la liste.

Le panneau des préférences de l'historique vous permet de configurer 3 paramètres du navigateur.

&pref.history.caption; :
&expireDaysMin1.label; __ &expireDaysMin2.label; :
Saisissez le nombre de jours pendant lequel vous voulez que &brandShortName; conserve une référence vers les pages Web que vous avez précédemment visitées. Par exemple, si vous mettez ce nombre à 10 jours, les pages visitées durant les 10 derniers jours seront conservées dans l'historique.
&clearHistory.label; :
Cliquez sur ce bouton pour effacer la liste des sites visités.
&locationBarHistory.caption; :
&clearLocationBarButton.label; :
Cliquez sur ce bouton pour effacer la liste des sites du menu de la barre d'adresse.

Pour plus d'information concernant l'historique dans &brandShortName;, voir la section Revenir aux pages précédentes.

Préférences du navigateur - &pref.applications.title;

Cette section décrit comment utiliser le panneau des préférences des applications. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Dans la catégorie &navigator.label;, cliquez sur &pref.applications.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &navigator.label; pour développer la liste.

Le panneau des préférences des applications permet de définir la façon dont les différents types de fichiers seront ouverts dans &brandShortName; par les autres applications :

The Helper Applications preferences panel lets you choose applications and other handlers to handle different types of content (e.g. PDF documents). It shows you a list of content types and lets you select a handler for each type. To filter the list, use the search field. Text entered in there will narrow the list to entries containing that text either in the type description or the currently selected action.

You can choose a local application to handle any type. For some types, you can also choose a web application to handle the type, choose a plugin in &brandShortName; to handle the type, or save the type on your computer.

To choose a handler for a type, select the type from the list. The current handler for the type will turn into a menu. Open the menu and select the handler you want to handle the type. Depending on the actual type, you can:

  • Choose an application by selecting it from the menu. If you want a local application that is not in the menu to handle the type, select &browse.label; from the menu and navigate to its location.
  • Choose a plugin or an application by selecting it from the menu.
  • Save files of this type on your computer by selecting &saveToDisk.label; from the menu. If you have selected the &autoDownload.label; preference in the &pref.download.title; panel, &brandShortName; will save content of this type on your computer automatically. Otherwise, when you encounter this type, &brandShortName; will prompt you for a location on your computer to save it to.
  • Tell &brandShortName; to &askBeforeOpen.label; what to do when encountering this type. When you choose this option, a dialog will always be shown when files of this type are accessed, and you can choose how to handle that specific file from there.

Pour plus d'informations sur la gestion des différents types de fichiers dans &brandShortName;, voir la section Plugins et téléchargements.

Préférences du navigateur - &pref.locationBar.title;

Cette section décrit comment utiliser le panneau des préférences de la barre de navigation. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Dans la catégorie &navigator.label;, cliquez sur &pref.locationBar.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &navigator.label; pour développer la liste.

Le panneau des préférences de la barre de navigation vous permet une meilleure navigation dans les pages Web en utilisant notamment l'auto-complétion de la barre d'adresse.

&autoComplete.label; :
&enableAutoComplete.label; :
Sélectionnez cette option pour laisser &brandShortName; afficher automatiquement les suggestions depuis votre historique de navigation lorsque vous écrivez dans la barre d'adresse.
  • &autoCompleteMatchOnlyTyped.label; : affiche uniquement les sites que vous avez saisis dans la barre d'adresse et non ceux qui auraient été ouverts d'une autre façon, tel qu'un clic sur un lien d'une page Web.
  • &autoCompleteAutoFill.label; : Lors de la saisie d'un texte dans la barre d'adresse, &brandShortName; complètera automatiquement avec les adresses qui correspondent au mieux à votre recherche.

    Remarque : Activer cette option préremplira les adresses locales (comme les chemins vers des fichiers de vitre ordinateur, même si vous avez précédemment désactivé l'option&enableAutoComplete.label;.

  • Montrer le moteur de recherche Internet : affiche un élément dans la liste déroulante vous permettant de faire des recherches pour les mots que vous entrez avec le moteur de recherche par défaut.
  • &autoCompleteShowPopup.label;: Lors de la saisie dans la barred'adresse, &brandShortName; affichera une liste déroulante des adresses Web visitées correspondantes.

    Note: Si vous avez désactivé l'option &enableAutoComplete.label;, les résultats correspondants de votre ordinateur seront toujors afficher dans la liste déroulante.

&searchEngine.label; :
&showSearchEngine.label; : Shows a drop-down list item, allowing you to search with the default search engine for words you enter.
&unknownLocations.label;
&domainGuessing.label; :
Sélectionnez cette option si vous désirez que &brandShortName; ajoute seul le préfixe www. et le suffixe .com lorsque vous saisissez une adresse. Pour plus d'informations sur cette fonctionnalité, consultez Domain Guessing.
&keywords.label;
Sélectionnez cette option pour permettre à &brandShortName; de lancer une lancer automatiquement une recherche internet pour tous texte saisi dans la barre d'adresse. Si le texte sais n'est pas une adresse Web, &brandShortName; efectuera une recherche internet lorsque vous appuyerez sur EntréeRetour dans la barre d'adresse. Consultez le document Mots clés Internet pour une description technique de cette fonctionnalité. Remarque : Le moteur de recherche utilisé ne peut pas être modifié par les Préférences de recherche sur Internet.

Préférences du navigateur - &tabHeader.label;

Cette section décrit comment utiliser le panneau des préférences des onglets. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Dans la catégorie &navigator.label;, cliquez sur &tabHeader.label;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &navigator.label; pour développer la liste.

Le panneau des préférences des onglets permet de configurer la navigation par onglets :

&tabDisplay.label; :
&autoHide.label; :
Sélectionnez cette option pour afficher une page Web sans la barre d'onglets quand il n'y en a qu'un seul d'ouvert.
&background.label; :
Désélectionnez cette option pour empêcher &brandShortName; de basculer vers le nouvel onglet quand on active l'option &openTab.label; dans &openTabs.label; pour ouvrir un lien.
&loadGroup.label; :
&loadGroupAppend.label;
Sélectionnez cette option si vous voulez conserver les onglets actuellement ouverts dans la fenêtre du navigateur.
&loadGroupReplace.label;
Sélectionnez cette option si vous voulez que les nouvelles pages s'ouvrent dans les onglets actuellement ouverts. Cette option vous évite d'avoir trop d'onglets à gérer en même temps.
&openTabs.label; :
Un clic sur la molette (ou le bouton central) ou un CmdCtrl+clic sur un lien d'une page Web :
Sélectionnez cette option pour ouvrir un lien d'une page Web dans un nouvel onglet quand on clique sur un lien avec la molette (ou le bouton du milieu) ou en pressant la touche CmdCtrl.
CmdCtrl+RetourEntrée dans la barre d'adresse
Sélectionnez cette option pour ouvrir une page Web dans un nouvel onglet lorsque vous tapez une adresse dans la barre d'adresses et que vous pressez ensuite sur CmdCtrl+RetourEntrée.
&newWindow.label; :
Si une page Web est conçue pour que certains liens s'ouvrent par défaut dans une nouvelle fenêtre (soit par l'attribut HTML target, soit à l'aide d'un script JavaScript), vous pouvez modifier ce comportement :
&openCurrent.label;
Ouvre la page liée dans l'onglet en cours de la fenêtre active.
&openTab.label;
Ouvre la page liée dans un nouvel onglet plutôt que dans une nouvelle fenêtre.
&openWindow.label;
Ouvre la page liée dans une nouvelle fenêtre (c'est l'option par défaut et qui ne change pas le comportement d'ouverture des liens).
&external.label; 
Si &brandShortName; est appelé à partir d'une autre application avec une adresse de page Web comme argument (comme par exemple cliquer sur un lien dans client de messagerie externe), vous pouvez contrôler comment sera chargée la page :
&openCurrent.label;
Ouvre la page liée dans l'onglet en cours de la fenêtre active.
&openTab.label;
Ouvre la page liée dans un nouvel onglet plutôt que dans une nouvelle fenêtre.
&openWindow.label;
Ouvre la page liée dans une nouvelle fenêtre (c'est l'option par défaut et qui ne change pas le comportement d'ouverture des liens).

Pour s'assurer que &brandShortName; ouvre une nouvelle fenêtre, sélectionnez l'option Navigateur dans les Préférences du navigateur - &tabHeader.label;.

Si vous voulez que &brandShortName; ouvre un nouvel onglet plutôt qu'une nouvelle fenêtre quand vous le lancez et qu'il est déjà démarré, faites ce qui suit :

  • Si vous démarrez &brandShortName; en ligne de commande, n'utilisez pas les paramètres de ligne de commande qui ouvre une fenêtre.
  • Rien n'est paramétré dans les Préférences du navigateur - &tabHeader.label;.
  • Cochez dans &external.label; l'option &openTab.label;.

Préférences du navigateur - &pref.download.title;

Cette section décrit comment utiliser le panneau des préférences des téléchargements. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Dans la catégorie &navigator.label;, cliquez sur &pref.download.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &navigator.label; pour développer la liste.

Le panneau des préférences des téléchargements vous permet de définir la façon dont &brandShortName; va traiter les informations sur les fichiers que vous téléchargez depuis une page Web

&downloadBehavior.label; :
&openDM.label;
Sélectionnez cette option pour ouvrir le gestionnaire de téléchargements, qui affiche l'état des téléchargements courants et précédents. L'état de tous les téléchargements est affiché dans une seule et même fenêtre.
&openProgressDialog.label;
Sélectionnez cette option pour ouvrir une boîte de dialogue qui vous renseignera sur la progression du téléchargement en cours. L'état de chaque téléchargement est indiqué dans des fenêtres séparées.
&doNothing.label;
Sélectionnez cette option si vous ne désirez pas afficher d'information sur les téléchargements.
&downloadLocation.label;
&promptDownload.label;
Sélectionnez cette option si voulez toujours être en mesure de choisir un répertoire pour le fichier à télécharger. Choisissez l'un des dossiers suivants comme emplacement par défaut pour le fichier :
&lastLocation.label;
L'emplacement par défaut sera le dernier répertoire où vous avez téléchargé un fichier.
&specifiedLocation.label;
L'emplacement par défaut sera le répertoire indiqué.
&autoDownload.label;
Sélectionnez cette option si vous voulez que vos fichiers soient enregistrés dans le répertoire courant sans que &brandShortName; vous demander d'indiquer l'emplacement.
&downloadFolder.label;
Utilisez le bouton &chooseDownloadFolder.label; pour vous rendre vers l'emplacement que vous désirez.
&finishedBehavior.label;
&playSound.label;
Sélectionnez cette option si vous voulez que &brandShortName; joue un son personnalisé (typiquement un fichier .WAV) quand un téléchargement est terminé. Utilisez le bouton &browse.label; pour choisir le fichier son. Pour écouter le son que vous avez choisi, cliquez sur le bouton &preview.label;.
&showAlert.label;
Sélectionnez cette option si vous voulez que &brandShortName; affiche une fenêtre d'avertissement lorsqu'un téléchargement est terminé.

Pour plus d'informations sur la façon de télécharger des fichiers depuis une page Web, voir la section Plugins et téléchargements.
