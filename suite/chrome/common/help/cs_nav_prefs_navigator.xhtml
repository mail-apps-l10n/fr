Préférences du navigateur - &navigator.label;

Cette section décrit comment utiliser le panneau principal des préférences du navigateur. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionner &preferencesCmd.label;.
  2. Sélectionnez la catégorie &navigator.label;.

Le panneau des préférences du navigateur vous permet de personnaliser certains aspects du navigateur.

&navRadio.label; :
Choisissez dans la liste déroulante l'élément dont vous voulez configurer l'ouverture :
  • &navStartPageMenu.label; ;
  • &newWinPageMenu.label; ;
  • &newTabPageMenu.label;.

Sélectionnez ensuite un de ces trois boutons pour déterminer ce que doit afficher l'élément lors de son ouverture :

&blankPageRadio.label; :
Le navigateur s'ouvre sans chargement automatique de page Web.
&homePageRadio.label; :
Le navigateur charge une page d'accueil (spécifiée ci-dessous) chaque fois qu'il est lancé.
&lastPageRadio.label; :
Le navigateur démarre en chargeant la dernière page visitée lors du dernier arrêt de &brandShortName;.
&restoreSessionRadio.label;
La navigateur restaurera les fenêtres et les onglets ouverts avant la dernière fermeture de &brandShortName;, y compris les données de formulaires et l'historique de navigation. Cette option n'est disponible qu'en complément de &navRadio.label; &navStartPageMenu.label;.
Navigateur par défaut :
Vous permet de définir &brandShortName; comme navigateur par défaut. Cette option n'est disponible que pour la version Microsfot® Windows de &brandShortName;.
&header2.label; :
Saisissez l'adresse de la page Web que vous voulez utiliser en page d'accueil, ou cliquez sur un des boutons suivants :
&useCurrent.label; :
Cliquez ici pour choisir la page Web en cours d'affichage sur le navigateur.
&useCurrentGroup.label; :
Si vous avez plusieurs onglets ouverts, ce bouton vous permet de définir ceux-ci comme pages d'accueil à lancer au démarrage de &brandShortName;. Après avoir cliqué sur ce bouton, vous verrez s'afficher le message Le groupe de pages d'accueil est fixé. Pour plus d'informations concernant l'ouverture d'onglets, voir la section Navigation par onglets.

Attention : Si vous éditez le champ texte après avoir appuyé sur le bouton &useCurrent.label;, vous perdrez le groupe d'onglets à afficher au démarrage.

&browseFile.label; :
Cliquez sur ce bouton pour rechercher sur votre disque dur, un fichier que vous voulez charger pour votre page d'accueil.
&useDefault.label; :
Cliquez sur ce bouton pour rétablir la page d'accueil par défaut de &brandShortName;.

&locationBarHistory.caption; :

&formfillHistory.caption;

Sélectionnez cette option pour autoriser &brandShortName; à conserver l'historique des formulaires que vous avez remplis ainsi que l'historique des recherches effectuées.

Le panneau des préférences des langues permet de choisir la langue et l'encodage des caractères pour l'affichage des pages Web lorsqu'elles sont disponibles dans plusieurs langues ainsi que l'activation ou non de la vérification orthographique à la saisie :

Préférences du navigateur - &pref.applications.title;

Cette section décrit comment utiliser le panneau des préférences des applications. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Dans la catégorie &navigator.label;, cliquez sur &pref.applications.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &navigator.label; pour développer la liste.

Le panneau des préférences des applications permet de définir la façon dont les différents types de fichiers seront ouverts dans &brandShortName; par les autres applications :

Le panneau des préférences des applications auxiliaires vous permet de choisir le comportement de &brandShortName; suivant le types de fichiers rencontrés (par exemple, les documents PDF). Il affiche une liste des types de contenus et vous permet de choisir le comportement pour chacun d'entre eux. Pour filtrer sur un type particulier, utilisez le champs de recherche. Le texte saisie dans ce champ réduira la liste aux entrées contenant ce texte soit dans la désignation du type de contenu soit dans l'action à effectuer définie pour un type de contenu.

Vous pouvez choisir une application locale pour ouvrir n'importe quel type de fichier. Pour certains types, vous pouvez également choisir une application Web, choisir un un plugin dans &brandShortName;, ou enregistrer ces types de fichiers sur votre ordinateur.

Pour choisir le comportement de &brandShortName; lorsque vous rencontrez un type de fichier, sélectionnez l'action à effectuer dans la liste déroulante en face du type. Selon le type de fichier, vou pouvez :

  • Choisir une application en la sélectionnant depuis le menu. Si vous voulez utiliser une application locale qui n'est pas répertorirée dans le menu, sélectionnez &browse.label; pour naviguer jusqu'à son emplacement.
  • Choisir un plugin ou une application depuis le menu.
  • Enregistrer les fichiers de ce type sur votre ordinateur en sélectionnant Enregistrer le fichier depuis le menu. Sinon, lorsque vous rencontrez ce type &brandShortName; vous invitera à choisir un emplacement pour l'enregistrer
  • Dire à &brandShortName; de Toujours demander l'action à effectuer. Si vous sélectionnez cette option, une boîte de dialogue s'affichera lorsque vous rencontrerez ce type de fichiers et vous pourrez choisir comment traiter ce fichier spécifique.

Pour plus d'informations sur la gestion des différents types de fichiers dans &brandShortName;, voir la section Plugins et téléchargements.

Préférences du navigateur - &pref.locationBar.title;

Cette section décrit comment utiliser le panneau des préférences de la barre de navigation. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Dans la catégorie &navigator.label;, cliquez sur &pref.locationBar.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &navigator.label; pour développer la liste.

Le panneau des préférences de la barre de navigation vous permet une meilleure navigation dans les pages Web en utilisant notamment l'auto-complétion de la barre d'adresse.

&autoComplete.label; :
&enableAutoComplete.label; :
Sélectionnez cette option pour laisser &brandShortName; afficher automatiquement les suggestions depuis votre historique de navigation lorsque vous écrivez dans la barre d'adresse.
&autoCompleteMatchOnlyTyped.label;
affiche uniquement les sites que vous avez saisis dans la barre d'adresse et non ceux qui auraient été ouverts d'une autre façon, tel qu'un clic sur un lien d'une page Web.
&autoCompleteMatchOnlyURLs.label;
Affiche uniquement les sites Web dont l'adresse correspond à la recherche. Les sites Web dont le titre correspond à ce que vous avez saisi ne s'afficheront pas dans les suggestions d'autocomplétion sauf si leur adresse correspond également.
&autoCompleteMatch.label;
&autoCompleteMatchAnywhere;
Les suggestions d'autocomplétion comprendront tous les sites Web que vous avez manuellement saisis pour lesquelles une partie de l'adresse ou du titre correspond à votre recherche.
&autoCompleteMatchWordsFirst;
Les suggestions d'autocomplétion comprendront tous les sites Web pour lequels n'importe quelle partie de l'adresse ou du titre correspond à votre recherche mais en préférant les résultats dont les correspondances se trouvent aux extrémités (voir ci-dessous). Ceci est la règle par défaut.
&autoCompleteMatchWords;
Les suggestions d'autocomplétion comprendront tous les sites Web pour lesquels votre recherche correspond au début de n'importe quel mot contenu dans l'adresse ou le titre. Des correspondances peuvent également être trouvées en milieu de mot si celui-ci contient des lettres majuscules (comme dans CamelCase) car tous les caractères majuscules sont traités comme des extrémités de mots.
&autoCompleteMatchStart;
Les suggestions d'autocomplétion comprendront tous les sites Web pour lesquels le début de l'adresse ou du titre correspond à votre recherche.
&autoCompleteAutoFill.label; 
Lors de la saisie d'un texte dans la barre d'adresse, &brandShortName; complètera automatiquement avec les adresses qui correspondent au mieux à votre recherche.

Remarque : Activer cette option préremplira les adresses locales (comme les chemins vers des fichiers de votre ordinateur, même si vous avez précédemment désactivé l'option&enableAutoComplete.label;.

Montrer le moteur de recherche Internet
affiche un élément dans la liste déroulante vous permettant de faire des recherches pour les mots que vous entrez avec le moteur de recherche par défaut.
&autoCompleteShowPopup.label;
Lors de la saisie dans la barre d'adresse, &brandShortName; affichera une liste déroulante des adresses Web visitées correspondantes.

Note: Si vous avez désactivé l'option &enableAutoComplete.label;, les résultats correspondants de votre ordinateur seront toujours affichés dans la liste déroulante.

&searchEngine.label; :
&showSearchEngine.label; : affiche une liste déroulante vous permettant d'effectuer une recherche sur le terme saisi à l'aide du moteur de recherche par défaut.
&unknownLocations.label;
&domainGuessing.label; :
Sélectionnez cette option si vous désirez que &brandShortName; ajoute seul le préfixe www. et le suffixe .com lorsque vous saisissez une adresse. Pour plus d'informations sur cette fonctionnalité, consultez Domain Guessing.
&keywords.label;
Sélectionnez cette option pour permettre à &brandShortName; de lancer automatiquement une recherche internet pour tout texte saisi dans la barre d'adresse. Si le texte saisi n'est pas une adresse Web, &brandShortName; effectuera une recherche internet lorsque vous appuyerez sur EntréeRetour dans la barre d'adresse. Consultez le document Mots clés Internet pour une description technique de cette fonctionnalité. Remarque : Le moteur de recherche utilisé ne peut pas être modifié par les Préférences de recherche sur Internet.

Préférences du navigateur - Comportement des liens

This section describes how to use the Link Behavior preferences panel. If you're not already viewing it, follow these steps:

  1. Open the &brandShortName; Edit menu and choose Preferences.
  2. Under the Browser category, click Link Behavior. (If no subcategories are visible, double-click Browser to expand the list.)

The Link Behavior preferences panel allows you to configure how links are opened in &brandShortName;:

&newWindow.label; :
Si une page Web est conçue pour que certains liens s'ouvrent par défaut dans une nouvelle fenêtre (soit par l'attribut HTML target, soit à l'aide d'un script JavaScript), vous pouvez modifier ce comportement :
&openCurrent.label;
Ouvre la page liée dans l'onglet en cours de la fenêtre active.
&openTab.label;
Ouvre la page liée dans un nouvel onglet plutôt que dans une nouvelle fenêtre.
&openWindow.label;
Ouvre la page liée dans une nouvelle fenêtre (c'est l'option par défaut et qui ne change pas le comportement d'ouverture des liens).
&external.label; 
Si &brandShortName; est appelé à partir d'une autre application avec une adresse de page Web comme argument (comme par exemple cliquer sur un lien dans client de messagerie externe), vous pouvez contrôler comment sera chargée la page :
&openCurrent.label;
Ouvre la page liée dans l'onglet en cours de la fenêtre active.
&openTab.label;
Ouvre la page liée dans un nouvel onglet plutôt que dans une nouvelle fenêtre.
&openWindow.label;
Ouvre la page liée dans une nouvelle fenêtre (c'est l'option par défaut et qui ne change pas le comportement d'ouverture des liens).

Pour s'assurer que &brandShortName; ouvre une nouvelle fenêtre, sélectionnez l'option Navigateur dans les Préférences du navigateur - &tabHeader.label;.

Si vous voulez que &brandShortName; ouvre un nouvel onglet plutôt qu'une nouvelle fenêtre quand vous le lancez et qu'il est déjà démarré, faites ce qui suit :

  • If &brandShortName; is launched from the command line, no command line parameters that open windows (like -new-window) are used.
  • No When &brandShortName; starts up, open option is set in Préférences du navigateur - &tabHeader.label;.
  • The Links from other applications preference is set to A new tab in the current window.

Préférences du navigateur - &pref.download.title;

Cette section décrit comment utiliser le panneau des préférences des téléchargements. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Dans la catégorie &navigator.label;, cliquez sur &pref.download.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &navigator.label; pour développer la liste.

Le panneau des préférences des téléchargements vous permet de définir la façon dont &brandShortName; va traiter les informations sur les fichiers que vous téléchargez depuis une page Web

&downloadBehavior.label; :
&focusWhenStarting.label;
Select this to change focus to the Download Manager if it is currently open. It displays the status for current and previous downloads in a single window. This option only takes effect if the Download Manager is already open and overrides the other settings.
&openDM.label;
Sélectionnez cette option pour ouvrir le gestionnaire de téléchargements, qui affiche l'état des téléchargements courants et précédents. L'état de tous les téléchargements est affiché dans une seule et même fenêtre.
&openProgressDialog.label;
Sélectionnez cette option pour ouvrir une boîte de dialogue qui vous renseignera sur la progression du téléchargement en cours. L'état de chaque téléchargement est indiqué dans des fenêtres séparées.
&doNothing.label;
Sélectionnez cette option si vous ne désirez pas afficher d'information sur les téléchargements.
&downloadLocation.label;
&alwaysAsk.label;
Sélectionnez cette option si voulez toujours être en mesure de choisir un répertoire pour le fichier à télécharger. Choisissez l'un des dossiers suivants comme emplacement par défaut pour le fichier :
&saveTo.label;
Utilisez le bouton &chooseDownloadFolder.label; pour vous rendre vers l'emplacement que vous désirez. &brandShortName; enregistrera automatiquement les fichiers téléchargés dans ce répertoire.
&downloadHistory.label;
&removeEntries.label;
Cette option détermine si l'entrée d'un téléchargement terminé doit être supprimée de la liste du gestionnaire de téléchargements (le fichier téléchargé ne sera pas affecté par cette suppression) :
&whenCompleted.label;
Sélectionnez cette option pour supprimer l'entrée immédiatement après la fin du téléchargement.
&whenQuittingApp.label;
Sélectionnez cette option pour conserver toutes les entrées de téléchargements jusqu'à la fermeture de &brandShortName;. Au prochain démarrage, la liste sera vide.
&neverRemove.label;
Sélectionnez cette option pour ne supprimer aucune entrée automatiquement, même après le démarrage de &brandShortName;. Vous pouvez supprimer les entrées manuellement de la liste du gestionnaire de téléchargements.
&finishedBehavior.label;
&playSound.label;
Sélectionnez cette option si vous voulez que &brandShortName; joue un son personnalisé (typiquement un fichier .WAV) quand un téléchargement est terminé. Utilisez le bouton &browse.label; pour choisir le fichier son. Pour écouter le son que vous avez choisi, cliquez sur le bouton &preview.label;.
&showAlert.label;
Sélectionnez cette option si vous voulez que &brandShortName; affiche une fenêtre d'avertissement lorsqu'un téléchargement est terminé.

Pour plus d'informations sur la façon de télécharger des fichiers depuis une page Web, voir la section Plugins et téléchargements.
