Raccourcis claviers de &brandShortName;

Ce document donne la liste de tous les raccourcis clavier que vous pouvez utiliser, en parallèle avec la souris, pour effectuer plus rapidement certaines opérations.

Dans cette section :

  • Utiliser les raccourcis

Utiliser les raccourcis

Ce document utilise les conventions suivantes pour présenter les raccourcis clavier de &brandShortName; :

Exemple
Commande Windows Mac OS Linux ou Unix
Copier Ctrl+C Cmd+C Ctrl+C

Pour exécuter une commande, appuyez simultanément sur toutes les touches indiquées. Par exemple, pour copier, sous ms-Windows, appuyez sur Ctrl et C en même temps.

Astuce : le signe plus (+) signifie que vous devez appuyer simultanément sur les deux touches indiquées. Dans le cas où il y aurait plus d'un signe plus (tel que dans Ctrl+Maj+C), vous devrez appuyer simultanément sur toutes les touches énumérées.

Nom des touches :

  • Ctrl = touche Contrôle.
  • Cmd = touche Commande ou Pomme sur un clavier Macintosh.
  • Début ou Origine = touche début (flèche dirigée vers le haut et la gauche, se trouvant au dessus de la touche Fin).
  • PgPréc/PgSuiv = ces touches peuvent être représentées par une flèche barrée dirigée vers le haut/bas.

Remarque : Certains raccourcis clavier exécutent des actions différentes suivant la position du curseur (zone de saisie active). Par exemple, si, sous ms-Windows, vous appuyez sur la touche Début alors que vous visualisez une page Web, &brandShortName; se placera en haut de celle-ci. Cependant, si vous effectuez cette même opération alors que le curseur est dans un champ texte, il se placera au début de ce champ.

Attention : Suivant votre clavier, certaines touches peuvent avoir des inscriptions différentes (par exemple la touche Echap pourra se nommer Esc, etc.).
