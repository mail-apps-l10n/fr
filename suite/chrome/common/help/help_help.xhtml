Utiliser la fenêtre d'Aide

Cette section décrit comment utiliser la fenêtre d'aide (c'est-à-dire la fenêtre contenant ce texte ainsi que les onglets situés à gauche).

Pour des sources additionnelles de support et d'information, cliquez sur les liens affichés dans la partie inférieure de l'accueil de l'aide.

Trouver une rubrique

  • Les principaux thèmes sont répertoriés dans le panneau Sommaire, à gauche. Cliquez sur le titre d'une rubrique pour lire son contenu. Pour voir dérouler les sous-sections d'un rubriques, cliquez sur le signe +triangle à gauche du nom de la rubrique principale.

Les liens qui figurent dans l'Aide s'utilisent comme ceux d'une page Web.

Par exemple, ce lien vous amène à une définition du glossaire : cookie. Utilisez les boutons &backButton.label; et &forwardButton.label; pour naviguer entre les sections que vous avez visitées.

Naviguer dans l'Aide

Pour naviguer dans l'Aide, utilisez les boutons de la barre de navigation de la fenêtre d'Aide :

  • Cliquez sur le bouton &backButton.label; pour retourner sur les pages que vous avez visitées précédemment. Il fonctionne comme le bouton &backButton.label; du navigateur.
  • Cliquez sur le bouton &forwardButton.label; pour aller sur les pages suivantes que vous avez visitées précédemment. Il fonctionne comme le bouton &forwardButton.label; du navigateur.
  • Cliquez le bouton &homeButton.label; pour revenir à la page d'accueil de l'Aide, incluant des liens vers des options de support et des ressources Web.

Vous pouvez aussi imprimer les instructions que vous souhaitez suivre :

  • Pour imprimer l'intégralité de la page de l'Aide affichée, cliquez le bouton &printButton.label; de la barre de navigation de la fenêtre d'Aide :


    &printButton.label;

  • Pour imprimer uniquement une partie de la page, sélectionnez la zone que vous souhaitez imprimer avec la souris, puis cliquez le bouton &printButton.label;.

Utiliser des raccourcis

Vous pouvez utiliser des raccourcis clavier pour naviguer dans la fenêtre d'Aide. Pour plus de détails, voir la section Raccourcis clavier de l'Aide.

Utiliser les boutons Aide

Beaucoup de fenêtres et boîtes de dialogue de &brandShortName; disposent d'un bouton appelé Aide.

Vous pouvez cliquer dessus pour ouvrir directement la fenêtre d'Aide à la section correspondante.

Astuces de recherche dans l'Aide

Vous ne trouvez pas ce que vous cherchez dans la recherche de l'aide ? Voici quelques astuces qui pourraient vous aider :

  • Assurez-vous de chercher quelque chose qui se rapporte à &brandShortName;. Il ne s'agit pas d'une recherche sur le Web.
  • Essayez d'élargir votre recherche, ne la restreignez pas trop. La formulation de l'aide pourrait être différente de ce que vous cherchez.
  • Évitez d'être trop large dans votre recherche. Des mots tels que Courrier ou &brandShortName; pourraient retourner toutes les pages de l'Aide.

Les différents liens de l'Aide

L'Aide en ligne de &brandShortName; vous permet de naviguer parmi les différentes sections à la manière d'un site Internet. Vous trouverez ainsi trois familles de liens :

Des liens simples
Ces liens pointent vers d'autres sections de l'Aide.
Des liens vers Internet
Ces liens sont aisément reconnaissables. Ils sont précédés de l'icône . Ils sont également suivis, entre crochets, par l'indication de la langue de la page Web (par exemple : [fr] pour une page en français, [en] pour une page en anglais, etc.).
Des liens en vert
Vous trouverez parfois des liens en vert. Ces liens renvoient à un glossaire qui définit brièvement les termes concernés.
Certains mots peuvent être soulignés de cette manière.
Il ne s'agit pas de lien à proprement parler. Si vous passez le curseur de la souris dessus, celui-ci se transforme en point d'interrogation. Vous verrez alors apparaître une info-bulle, décrivant le sigle, l'abréviation, etc.
