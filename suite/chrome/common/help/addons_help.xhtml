Les modules complémentaires

Un des moyens les plus excitant pour personnaliser &brandShortName; est d'utiliser les modules complémentaires. Dans cette section, vous apprendrez ce que sont les modules complémentaires, comment les installer et comment utiliser le gestionnaire de modules.

À propos des modules complémentaires

Un module complémentaire est un petit programme qui peut être ajouté à &brandShortName; pour modifier sont apparence, son comportement ou pour ajouter de nouvelles fonctionnalités. Il peut également la langue de l'interface utilisateur. Les modules complémentaires sont classifiés selon 4 catégories :

Extensions
Ce type de module peut soit changer le comportement de &brandShortName; soit ajouter de nouvelle fonctionnalités. Actuellement, il existe des modules pour afficher les prévisions météo, pour inclure un client FTP ou encore pour bloquer les publicités des pages Web. Certains modules ne sont conçus que pour fonctionner avec des sites Web spécifiques.
Thèmes
Les thèmes modifie l'apparence et l'esthétisme de &brandShortName;, mais ne modifient pas ses fonctionnalités. &brandShortName; est fourni avec deux thèmes, le thème par défaut et le thème Modern, mais vous pouvez en ajouter beaucoup d'autres et utiliser ceux qui vous plaisent.
Languages
Bien que vous pouvez téléchargez &brandShortName; dans un grand nombre de langues, vous pouvez également lui en ajouter d'autres sous la forme de paquets de langues. De cette manière, différents utilisateurs peuvent utiliser &brandShortName; dans leur langue sans avoir à installer un programme pour chacun. Ceci est particulièrement utile si vous devez installer &brandShortName; sur un ordinateur partagé par plusieurs utilisateur, par exemple un ordinateur de laboratoire.

Remarque : les paquets de langues modifie uniquement la langue de l'interface utilisateur. Les pages Web et les messages électroniques restent dans la langues dans laquelle ils ont été écrit.

Plugins
Plugins agissent comme des connecteurs entre &brandShortName; et d'autres logiciels. Ils vous permettent de visionner des contenus spéciaux (comme des animations Flash ou des documents PDF) directement depuis une fenêtre de votre navigateur plutôt que d'ouvrir un logiciel externe.

Les modules complémentaires sont toujours fournis sous la forme de paquets dont l'extension est XPI, excepté dans le cas de plugins qui sont généralement des fichiers binaires spécifiques à un système d'exploitation.

Installer des modules complémentaires

Il y a plus façons d'installer un module complémentaire :

  • En cliquant sur le lien d'un module complémentaire dans une page Web, un bandeau apparait dans &brandShortName; pour vous demander de confirmer ou d'annuler l'installation. Le bouton d'installation est inactif pendant quelques secondes pour vous éviter de le cliquer accidentellement à l'apparition du bandeau.

    Remarque : comme mesure de sécurité supplémentaire, &brandShortName; ne permet l'installation de modules complémentaires qu'à partir d'une liste de sites autorisés. Cette liste peut être éditée depuis le panneau des préférences avancées Installation de logiciels.

  • En ouvrant un paquet de module précédemment enregistré sur votre ordinateur, comme pour l'ouverture d'un fichier normal. Sélectionnez Ouvrir un fichier… depuis le menu Fichier, ou appuyez sur la touche CmdCtrlO.
  • En utilisant le panneau Catalogue du gestionnaire de modules complémentaires ou en cliquant sur le bouton Installer depuis n'importe quel panneau du gestionnaire de modules.

Remarque : les plugins proviennent généralement de programmes à installer séparément lorsque &brandShortName; est fermé, afin que ce programme externe et le plugin correspondant à &brandShortName; s'installent correctement.

Utiliser le gestionnaire de modules

Le gestionnaire de modules complémentaires permet d'installer et ou désinstaller des modules, d'activer ou de désactiver un thème et dans certains cas de définir leurs préférences.

Pour ouvrir le gestionnaire de modules complémentaires, suivez une de ces étapes :

  • Ouvrez le menu Outils puis sélectionnez Modules complémentaires.
  • Ouvre le menu &brandShortName;Édition et sélectionnez Préférences. Développez la catégorie Avancé et cliquez sur Installation de logiciels.

Panneau du catalogue

La panneau Catalogue du gestionnaire de modules utilise des services Web fournit par le site Mozilla Add-ons afin de présenter une liste des modules recommandés ou de permettre d'effectuer une recherche parmi les modules disponible sur Mozilla Add-ons. Après avoir ouvert le gestionnaire de modules complémentaires, cliquez sur Catalogue pour afficher le panneau. Vous pouvez alors effectuer les actions suivantes :

  • Rechercher un module par un nom, une description ou une étiquette, à partir du champ de recherche. Saisissez un mot ou une phrase et appuyez sur RetourEntrée. Vous obtiendrez une liste des modules correspondants. Pour effacer les termes recherchés et la liste des résultats, cliquez sur l'icône à droite du champ de rechercher.

    Remarque : les modules déjà installés ou inompatibles avec votre version de &brandShortName; ne s'afficheront pas dans la liste des résultats.

  • Pour visiter le site Mozilla Add-ons depuis le navigateur, cliquez sur Parcourir tous les modules complémentaires.
  • Pour obtenir plus d'informations à propos d'un module recommandé, cliquez sur la ligne correspondante. Celle-ci se développera pour afficher une image et une description du module, ainsi que des indications concernant le type du module (extension, thème, etc.). Vous pouvez également installer le module en cliquant sur le bouton Ajouter à &brandShortName;….
  • Pour afficher la liste complète des modules recommandés pour &brandShortName; depuis le navigateur, allez en bas de la liste et cliquez sur le lien Voir tous les modules complémentaires recommandés.

Panneau des extensions

Le panneau des extensions du gestionnaire de modules complémentaires liste les extensions installées. Après avoir ouvert le gestionnaire de modules complémentaires, cliquez sur le bouton Extensions pour afficher le panneau. Vous pouvez alors effectuer les actions suivantes :

  • Pour obtenir plus d'informations à propos de n'importe quelle extension installé, cliquez sur l'entrée correspondante dans la liste. L'entrée se développera et affichera une description complète de l'extension.

    Remarque : les extensions désactivées seront grisées. Les extensions incompatibles seront grisées, car elles sont également désactivées et leur icône comporte un signe « Interdit » superposé (les extensions incompatibles sont celles qui se définissent elles-mêmes comme non compatible avec votre version de &brandShortName;).

  • Pour accéder aux préférences d'une extension (si elle possède un panneau de préférences), cliquez sur l'entrée correspondante à l'extension puis cliquez sur le bouton « Préfénreces ».

    Remarque : si l´extension ne possède pas de préférences, le bouton est désactivé. Vous ne pouvez pas accéder aux préférences des extensions désactivées ou incompatibles.

  • To disable an extension, click the entry corresponding to the desired extension, then click the Disable button. An info bar will appear to remind you that changes will apply once you restart &brandShortName;. The info bar features a Restart &brandShortName; button.
  • To enable an extension, click the entry corresponding to the desired extension, then click the Enable button. An info bar will appear to remind you that changes will apply once you restart &brandShortName;. The info bar features a Restart &brandShortName; button.
  • To uninstall an extension, click the entry corresponding to the desired extension, then click the Uninstall button. An info bar will appear to remind you that changes will apply once you restart &brandShortName;. The info bar features a Restart &brandShortName; button.
  • Pour chercher les mises à jours des extensions installées (y compris celles marquées comme incompatibles), cliquez sur le bouton « Rechercher des mises à jour » situé en bas du gestionnaire de modules complémentaires.

    Si des mises à jour sont disponibles, elles s'afficheront dans un nouveau panneau. Dans ce panneau, vous pourrez choisir quelles mises à jour appliquer. Cliquez alors sur le bouton « Installer les mises à jour » pour récupérer les modules mis à jour. Une fois téléchargées, vous devez redémarrer &brandShortName; pour appliquer les mises à jour.

Panneau des thèmes

Le panneau des thèmes du gestionnaire de modules complémentaires liste les thèmes installés. Après avoir ouvert le gestionnaire de modules complémentaires, cliquez sur le bouton Thèmes pour afficher le panneau. Vous pouvez alors effectuer les actions suivantes :

  • To get more information and a preview on any of the installed themes, click an entry in the list. The entry will expand and show the full description of the theme, along with a preview in the right side of the Add-on Manager dialog (you may need to resize the dialog to properly see the preview pane).
  • To use a theme, click the entry corresponding to the desired theme, then click the Use Theme button. An info bar will appear to remind you that changes will apply once you restart &brandShortName;. The info bar features a Restart &brandShortName; button.

    Note: Contrary to extensions, you only can use one theme at a time.

  • To uninstall a theme, click the entry corresponding to the desired theme, then click the Uninstall button. An info bar will appear to remind you that changes will apply once you restart &brandShortName;. The info bar features a Restart &brandShortName; button.

Panneau des langues

Le panneau des langues du gestionnaire de modules complémentaires liste les paquets de langues installés. Ce panneau ne s'affiche que si vous avez installé au moins un paquet de langue.

Après avoir ouvert le gestionnaire de modules complémentaires, cliquez sur le bouton Langues pour afficher le panneau. Vous pouvez alors effectuer les actions suivantes :

  • To disable a language pack, click the entry corresponding to the desired language pack, then click the Disable button. An info bar will appear to remind you that changes will apply once you restart &brandShortName;. The info bar features a Restart &brandShortName; button.
  • To enable a language pack, click the entry corresponding to the desired language pack, then click the Enable button. An info bar will appear to remind you that changes will apply once you restart &brandShortName;. The info bar features a Restart &brandShortName; button.
  • To uninstall a language pack, click the entry corresponding to the desired language pack, then click the Uninstall button. An info bar will appear to remind you that changes will apply once you restart &brandShortName;. The info bar features a Restart &brandShortName; button.

Remarque : l'activation d'un paquet de langue dans le panneau des langues ne change pas la langue de l'interface de &brandShortName;'. Elle sera disponible à partir du panneau Apparence des préférences où vous devrez sélectionner la langue à utiliser.

[ Retour au début de la section ]

Panneau des plugins

Le panneau des plugins liste tous les types de plugins installés et disponibles. Après avoir ouvert le gestionnaire de modules complémentaires, cliquez sur le bouton Plugins pour afficher le panneau. Vous pouvez alors effectuer les actions suivantes :

  • Désactiver un plugin en sélectionnant la ligne correspondante pour en cliquant sur le bouton Désactiver.
  • Activer un plugin en sélectionnant la ligne correspondante pour en cliquant sur le bouton Activer.

Remarque : contrairement aux autres types de modules, l'activation ou la désactivation d'un plugin ne nécessite pas le redémarrage de &brandShortName;.
