Outils de développement Web

&brandShortName; offre aux développeurs Web un certain nombre d'outils utiles lors du développement et du débogage d'un projet Web. Leur installation peut être optionnelle. Pour accéder à ces outils, ouvrez le menu Outils et choisissez Développement Web.

  • &errorConsoleCmd.label; : La console affiche les alertes, erreurs et autres messages concernant le code JavaScript et les feuilles de style CSS des pages ou applications Web, ainsi que sur &brandShortName; lui-même. Par défaut, les erreurs d'interprétation CSS et les erreurs JavaScript sont affichées. La console peut également être utilisée pour afficher des messages concernant les codes XUL et JavaScript.

    Ressources Web :

    • Utiliser &brandShortName; pour tester et déboguer les sites Web, par Mozinet.
  • &Inspector.title; : Cet outil sert à vérifier et à éditer le DOM des documents Web ainsi que des applications XUL.

    Ressources Web :

    • Introduction to the DOM Inspector (Ian Oeschger) ;
    • DOM Inspector FAQ (mozilla.org) ;
    • Petit guide d'initiation à l'Inspecteur DOM ;
    • Creating Applications with Mozilla - Appendix B3 : The DOM Inspector.
  • &venkmanCmd.label; : également connu sous le nom de Venkman, cet utilitaire vous permet de déboguer le JavaScript de vos projets Web depuis &brandShortName;. Il permet l'exécution pas à pas, les points d'arrêt et fournit de nombreuses autres fonctionnalités.

    Ressources Web :

    • Page du projet Venkman, débogueur de JavaScript
    • Learning the JavaScript debugger Venkman (Svend Tofte)
    • Debugging JavaScript Using Venkman (webreference.com, articles)
    • Venkman Development Homepage
  • Développement XUL : XUL est language dans lequel sont codés les logiciels de la fondation Mozilla ainsi que toutes les extensions disponibles.

    Ressources Web :

    • XULPlanet, regroupe les réféences aux fonctions XUL et tout un ensemble de didacticiels.
    • Mozilla Developer Center, XUL, XSL, CSS, XPCOM, JavaScript, documentation sur la compilation, sur les extensions, sur les thèmes, etc. Tous ce que vous avez toujours voulu savoir sur les technologies Mozilla sans jamais oser le demander.
    • XULfr.org, le site de référence francophone. XULfr regroupe l'ensemble des traductions du site anglais XULPlanet.
    • Les forums XULfr.org, pour poser vos questions à propos du développement d'extensions ou d'applications XUL.
