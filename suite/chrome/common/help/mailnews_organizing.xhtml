Organiser vos messages

Dans cette section :

  • Créer un dossier
  • Changer le nom d'un dossier
  • Déplacer ou copier un dossier
  • Archiver des messages dans des dossiers
  • Partager un dossier avec d'autres utilisateurs (IMAP seulement)
  • Étiqueter des messages
  • Marquer ou cocher des messages
  • Utiliser les modes d'affichage des messages
  • Créer des filtres de messages
  • Rechercher des messages

Créer un dossier

Pour créer un dossier de messagerie :

  1. Ouvrez une fenêtre Courrier.
  2. Ouvrez le menu &fileMenu.label;, choisissez &newMenu.label;, puis &newFolderCmd.label;. La boîte de dialogue Nouveau dossier s'affiche.
  3. Entrez le nom du dossier.
  4. Cliquez sur la liste déroulante pour choisir un emplacement et cliquez sur OK. Le nouveau dossier apparaît dans la liste des dossiers de messagerie.

[ Retour au début de la section ]

Changer le nom d'un dossier

Pour renommer un dossier existant :

  1. Ouvrez une fenêtre Courrier.
  2. Sélectionnez le dossier à renommer.
  3. Ouvrez le menu &fileMenu.label; et choisissez Renommer le dossier.
  4. Entrez le nouveau nom du dossier et cliquez sur OK.

Remarque : si vous renommez un dossier utilisé pour stocker des messages filtrés, le filtre sera automatiquement mis à jour afin d'utiliser le dossier renommé.

[ Retour au début de la section ]

Déplacer ou copier un dossier

Vous pouvez copier un dossier et son contenu dans un autre compte de messagerie ou le déplacer dans le même compte.

Pour déplacer ou copier un dossier :

  1. Ouvrez une fenêtre Courrier.
  2. Sélectionnez le dossier à déplacer ou à copier.
  3. Effectuez l'une des opérations suivantes :
    • Pour déplacer le dossier dans un autre dossier du même compte, faites-le glisser sur le nom du dossier en question. Le dossier déplacé devient un sous-dossier du dossier cible.
    • Pour copier le dossier dans un autre compte, faites-le glisser sur le nom de ce dernier.
    • Pour copier le dossier dans un autre dossier d'un autre compte, faites-le glisser sur le nom de ce dossier. Le dossier copié devient un sous-dossier du dossier cible.

Archiver des messages dans des dossiers

Vous pouvez déplacer des messages d'un dossier à un autre à l'aide des méthodes suivantes :

  • Sélectionnez le message, cliquez sur le bouton &fileButton.label; de la barre d'outils et choisissez le dossier de destination dans la liste déroulante.
  • Glissez-déplacez des messages vers le dossier désiré.

    Remarque : si vous glissez-déplacez un message d'un dossier de serveur de messagerie IMAP ou POP vers un dossier local sur le disque dur, il est placé dans le dossier local et supprimé du dossier du serveur.

Pour copier un message d'un dossier dans un autre :

  1. Sélectionnez le message et cliquez dessus avec le bouton droit pour afficher le menu contextuel.
  2. Sélectionnez Copier vers et sélectionnez le compte et le dossier de destination dans les sous-menus.

Astuce : d'une autre manière, vous pouvez copier un message entre dossiers en pressant sur la touche MAJ tout en glissant-déplaçant le message de la liste des messages sur un dossier.

Partager un dossier avec d'autres utilisateurs (IMAP seulement)

Les utilisateurs avec des comptes courrier IMAP peuvent partager des dossiers avec d'autres utilisateurs du même réseau. Partager des dossiers permet à plusieurs utilisateurs de voir et de travailler sur les mêmes messages, comme dans un groupe. Pour utiliser le partage de dossiers, votre serveur IMAP doit supporter la gestion des listes de contrôle d'accès (ACL, Access Control List). Demandez à votre administrateur système si vous n'êtes pas sûr que les dossiers partagés sont supportés par votre serveur IMAP.

Pour partager un dossier avec d'autres utilisateurs de votre réseau ou pour voir des informations partagées :

  1. Ouvrez une fenêtre Courrier.
  2. Dans un compte IMAP, sélectionnez un dossier que vous voulez partager ou un dossier dont vous voulez voir les privilèges de partage.

    Remarque : les dossiers dans les dossiers locaux ou dans un compte POP ne peuvent être partagés.

  3. Ouvrez le menu &editMenu.label; et choisissez &folderPropsFolderCmd.label;.
  4. Cliquez sur l'onglet Partage.
  5. Cliquez sur Privilèges. Vous pouvez avoir à saisir vos nom d'utilisateur et mot de passe.

    Le bouton Privilèges n'est disponible que si le serveur IMAP vous permet de modifier les privilèges de partage des dossiers. Si ce bouton n'est pas disponible, vous pouvez voir les privilèges de partage de ce dossier, mais pas les modifier.

  6. Suivez les instructions sur l'écran pour ajouter des utilisateurs et définir leurs droits d'accès aux dossiers.
    Privilèges de lecture :
    Les utilisateurs peuvent lire les messages et copier leur contenu, mais ne peuvent ni les modifier, ni les effacer, ni copier de nouveaux messages dans le dossier. Les utilisateurs peuvent marquer les messages comme lus ou non lus. Voir la section Étiqueter ou marquer des messages pour des détails sur le marquage des messages.
    Privilèges de lecture-écriture :
    En plus des privilèges de lecture, les utilisateurs peuvent modifier et supprimer des messages. Les utilisateurs peuvent aussi copier ou déplacer des messages dans le dossier.
    Privilèges d'administration :
    En plus des privilèges de lecture-écriture, les utilisateurs peuvent ajouter et supprimer des utilisateurs et changer leurs privilèges sur les dossiers.
  7. Cliquez sur OK pour confirmer vos modifications.
  8. Cliquez sur OK pour fermer la boîte de dialogue Propriétés du dossier.

Dans la liste des dossiers de votre compte de courrier, un dossier partagé se distingue par son icône.

Pour envoyer un message d'instructions d'accès à votre dossier partagé à partir d'une fenêtre Courrier :

  1. Sélectionnez le dossier partagé.
  2. Cliquez du bouton droit pour afficher un menu contextuel et choisissez Copier l'adresse du dossier.
  3. Cliquez sur Rédiger pour afficher une fenêtre de rédaction de message.
  4. Cliquez dans le corps du message, ouvrez le menu &editMenu.label; et choisissez &pasteCmd.label;.
  5. Remplissez les destinataires, ajoutez un sujet et tapez le texte du message. Avertissez les destinataires qu'ils peuvent accéder au dossier partagé en cliquant sur le lien collé dans le message.

    Seuls les destinataires sur le même réseau pourront accéder à votre dossier partagé.

  6. Cliquez sur Envoyer.

Accéder à un dossier partagé

Accéder à un dossier partagé est comparable à l'abonnement à un groupe. Pour accéder à un dossier partagé :

  1. Ouvrez une fenêtre Courrier .
  2. Ouvrez le menu &fileMenu.label; et choisissez &subscribeCmd.label;. La boîte de dialogue Abonnement apparaît.
  3. Le cas échéant, cliquez sur la liste déroulante Serveur pour choisir le bon compte de courrier IMAP.
  4. Sélectionnez le dossier auquel vous voulez vous abonner.
  5. Cliquez sur le bouton S'abonner ou sur la case proche du nom du dossier dans la colonne Abonné. Vous voyez un crochet dans la case près de chacun des dossiers auquel vous vous êtes abonné. Cliquez sur &unsubscribeNewsgroupCmd.label; pour annuler un abonnement.
  6. Cliquez sur OK. La liste des dossiers auxquels vous êtes abonné s'affiche dans la fenêtre Courrier.

Étiqueter des messages

Vous pouvez étiqueter des messages pour vous aider à les organiser et à leur donner des priorités. Vous pouvez également en créer de nouvelles et leur donner une couleur et un texte d'étiquette standards ou personnaliser ceux-ci pour répondre à vos besoins

Une application particulièrement intéressante est l'utilisation des filtres de messages pour étiqueter automatiquement les messages entrants. Par exemple, vous pouvez configurer un filtre de message pour qu'il étiquette les messages de votre patron comme Important et pour qu'ils apparaissent en rouge. Voir la section Créer des filtres de messages pour plus d'informations.

Étiquetage

Pour étiqueter un message :

  1. Ouvrez une fenêtre Courrier.
  2. Sélectionnez le message que vous désirez étiqueter.
  3. Ouvrez le menu &msgsMenu.label; et choisissez Étiquette.
  4. Choisissez dans la liste l'étiquetage que vous désirez employer.

L'en-tête du message passe dans la couleur d'étiquetage que vous avez choisie. Pour voir le texte de l'étiquette, vous devez afficher la colonne Étiquette dans la fenêtre Courrier.

Astuce : pour étiqueter rapidement des messages, sélectionnez-les et appuyez sur un chiffre de 1 à 9 sur votre clavier. Appuyez sur 0 pour enlever l'étiquette.

Pour faire apparaître la colonne Étiquette :

  • Ouvrez une fenêtre Courrier.
  • Cliquez sur l'icône Afficher/Masquer les colonnes et sélectionnez Étiquette dans la liste.

Remarque : les étiquettes s'appliquent indépendamment pour chaque compte. Par exemple, si vous déplacez ou copiez un message étiqueté vers un autre compte de messagerie, l'étiquette n'est pas préservée. De même si vous transférez un message étiqueté vers un autre destinataire. Dans le cas d'un compte de messagerie IMAP, si le serveur supporte les mots-clés définis par l'utilisateur, les étiquettes de messages resteront quand vous vous connecterez à votre compte de messagerie d'un autre endroit.

&pref.tags.caption;

Vous pouvez personnaliser la couleur, le texte ou l'ordre de vos étiquettes afin de les adapter à vos besoins.

Pour personnaliser les étiquettes :

  1. Ouvrez une fenêtre Courrier.
  2. Ouvrez le menu SeaMonkey&editMenu.label; et choisissez &preferencesCmd.label;. La boîte de dialogue &prefWindow.title; s'affiche.
  3. Sous la catégorie Courrier et Groupes, cliquez sur &pref.tags.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie Courrier et Groupes pour développer la liste.
  4. Cliquez dans le champ texte et modifiez le nom de l'étiquette en le remplaçant par votre propre texte. Le nom de l'étiquette peut contenir jusqu'à 32 caractères.
  5. Pour changer la couleur de l'étiquette, cliquez sur le bloc de couleur près du texte de l'étiquette et sélectionnez une nouvelle couleur.
  6. Cliquez les boutons « Monter » et « Descendre » pour réarranger l'ordre des étiquettes. La coloration des messages se fera préférentiellement à partir de l'ordre de la liste ?
  7. Cliquez sur OK.

Vos changements prendront immédiatement effet pour tous les messages étiquetés dans tous vos comptes de messagerie.

Astuce : pour restaurer les paramètres par défaut des étiquettes, procédez comme ci-dessus pour afficher les paramètres d'étiquetage puis cliquez sur Réinitialiser.

Trier les messages par étiquette

Pour trier les messages par étiquette :

  1. Ouvrez une fenêtre Courrier.
  2. Pour faire apparaître la colonne Étiquette si elle est cachée, cliquez sur l'icône Afficher/Masquer les colonnes et sélectionnez Étiquette dans la liste.
  3. Cliquez sur la colonne Étiquette pour trier les messages par couleur d'étiquette. Dans une même couleur, les messages sont ensuite triés par date.

Supprimer des étiquettes

Pour effacer une étiquette :

  1. Ouvrez une fenêtre Courrier.
  2. Sélectionnez un ou plusieurs messages étiquetés.
  3. Ouvrez le menu &msgsMenu.label; et choisissez Étiquette.
  4. Sélectionnez l'étiquette à supprimer ou Aucune pour les supprimer toutes.

Astuce : utiliser les raccourcis clavier pour attribuer des étiquettes à vos messages. Pour supprimer une étiquette, le raccourci clavier est 0.

Marquer ou cocher des messages

Il peut être pratique de marquer un message déjà lu comme non lu pour faciliter la relecture ou la rédaction d'une réponse.

Pour marquer un message comme non lu :

  1. Ouvrez une fenêtre Courrier.
  2. Sélectionnez un dossier de messagerie ou de groupe pour faire apparaître les messages.
  3. Cliquez sur la colonne Lu devant chaque message que vous désirez marquer comme non lu. Les messages marqués comme non lus affichent un symbole dans la colonne Lu et le titre du message s'affiche en gras. Les messages marqués comme lus affichent un symbole dans la colonne Lu. Pour faire apparaître la colonne Lu si elle est cachée, cliquez sur l'icône Afficher/Masquer les colonnes et sélectionnez Lu dans la liste.
Colonne Lu Colonne Coché

Vous pouvez cocher les messages que vous désirez télécharger plus tard pour une utilisation hors ligne.

Pour cocher des messages :

  1. Ouvrez une fenêtre Courrier.
  2. Sélectionnez un dossier de messagerie ou de groupe pour faire apparaître ses messages.
  3. Cliquez sur la colonne Coché de chaque message que vous désirez télécharger. Un drapeau apparaît lorsque vous cliquez pour indiquer que le message a été coché. Si la colonne Coché n'est pas visible, cliquez sur l'icône Afficher/Masquer les colonnes et sélectionnez Coché dans la liste.

Utiliser les modes d'affichage des messages

Vous pouvez appliquer un mode d'affichage prédéfini ou personnalisé pour vous aidez à gérer le courrier en filtrant les messages à ne pas afficher.

Pour utiliser un mode d'affichage des messages, ouvrez le menu &viewMenu.label; et sélectionnez Messages. Choisissez alors une option du sous-menu.

Tous :
Choisissez cette option pour afficher tous les messages.
Non lus :
Choisissez cette option pour n'afficher que les messages non lus.
Nom d'une étiquette :
Choisissez une étiquette pour n'afficher que les messages qui correspondent.
Affichage personnalisé :
Choisissez un mode d'affichage personnalisé. Par défaut, vous avez ces 5 modes : Personnes que je connais, Courriers récents, Derniers 5 jours, Non indésirables et Avec pièces jointes.
&viewCustomizeView.label;
Choisissez cette option pour voir ou modifier les paramètres des affichages personnalisés.

Astuce : vous pouvez changer rapidement le mode d'affichage des messages depuis la liste déroulante Voir dans la barre de recherche. Si vous ne voyez pas la barre de recherche :

  1. Ouvrez le menu &viewMenu.label;.
  2. Sélectionnez &viewToolbarsMenu.label;.
  3. Choisissez &showSearchToolbarCmd.label;.

Créer un mode d'affichage personnalisé

Vous pouvez créer des vues de messages personnalisées pour n'afficher que les messages répondant à certains critères.

Pour changer ou créer un mode d'affichage personnalisé, suivez ces étapes :

  1. Ouvrez le menu &viewMenu.label;, sélectionnez Messages, puis choisissez Personnaliser…
  2. Pour créer une nouvelle vue, cliquez sur Nouveau. Pour modifier une vue existante, sélectionnez la mode et cliquez sur Édition.
  3. Saisissez un nom pour ce mode d'affichage.
  4. Sélectionnez l'option correspondante pour la vue que vous désirez utiliser : valident toutes les conditions suivantes que vous ajouterez, ou valident au moins une des conditions suivantes.
  5. Utilisez les listes déroulantes pour choisir le critère de recherche (par exemple, Sujet, Expéditeur, contient, ne contient pas) puis saisissez les mots ou la phrase qui vous voulez chercher.

    Astuce : pour chercher des messages contenant un en-tête non listé dans la première liste déroulante (par exemple, si vous recherchez des messages qui incluent un champ Resent-From), choisissez Personnaliser et entrez l'en-tête dans laquelle vous voulez chercher. &brandShortName; Courrier & Groupes ajoutera le nouvel en-tête à la liste déroulante. Ainsi, vous pourrez le choisir pour d'autres règles de messages. Assurez-vous d'avoir correctement saisi le nom du champ, car &brandShortName; Courrier & Groupes ne peut trouver que des correspondances exactes.

  6. Cliquez sur Plus pour ajouter des critères et sur Moins pour en enlever.
  7. Cliquez sur OK pour confirmer vos changements.
  8. Cliquez sur OK dans la boîte de dialogue Personnaliser les vues des messages. Les paramètres de la vue sélectionnée s'appliquent automatiquement.

Créer des filtres de messages

Les filtres de messages permettent de gérer et d'organiser les messages automatiquement. Vous pouvez créer des filtres utilisés par &brandShortName; Courrier & Groupes pour effectuer automatiquement certaines actions sur les messages entrants en fonction de critères spécifiques. Par exemple, vous pouvez créer un filtre de messages qui déplacera automatiquement les messages entrants dans un dossier ou sous-dossier. Les filtres de messages sont conçus pour un compte particulier.

Pour créer un filtre de messages :

  1. Ouvrez une fenêtre Courrier.
  2. Ouvrez le menu &tasksMenu.label; et choisissez &filtersCmd.label;. La boîte de dialogue Filtres de messages s'affiche.
  3. Si vous avez plusieurs comptes de messagerie, sélectionnez celui sur lequel vous souhaitez appliquer le filtre.
  4. Cliquez sur Nouveau. La boîte de dialogue Modifier le filtre permet de spécifier les types de messages concernés et l'action à effectuer par le filtre.
  5. Entrez un nom pour le filtre.
  6. Sélectionnez l'option de condition qui convient au filtre que vous souhaitez appliquer : valident TOUTES les conditions suivantes ou valident au moins UNE des conditions suivantes.
  7. Utilisez les listes déroulantes pour sélectionner les critères de recherche (par exemple, Objet, Expéditeur, Contient, Ne contient pas) et entrez le texte à rechercher.

    Astuce : pour chercher des messages contenant un en-tête non listé dans la première liste déroulante (par exemple, si vous recherchez des messages qui incluent un champ Resent-From), choisissez Personnaliser et entrez l'en-tête dans laquelle vous voulez chercher. &brandShortName; Courrier & Groupes ajoutera le nouvel en-tête à la liste déroulante. Ainsi, vous pourrez le choisir pour d'autres règles de messages. Assurez-vous d'avoir correctement saisi le nom du champ, car &brandShortName; Courrier & Groupes ne peut trouver que des correspondances exactes.

  8. Cliquez sur Plus pour ajouter des critères et sur Moins pour supprimer le dernier critère de la liste.
  9. Cochez les cases pour sélectionner l'action (ou les actions) que vous souhaitez que le filtre applique aux messages (par exemple Déplacer vers le dossier). Si elles existent, utilisez les listes déroulantes en face de chaque option, afin de préciser l'action à exécuter.

    Astuce : pour étiqueter automatiquement les messages entrants, optez pour Étiqueter l'en-tête du message dans la liste déroulante.

  10. Cliquez sur OK pour confirmer les paramètres.
  11. Cliquez sur OK dans la boîte de dialogue Filtres de messages. Le filtre commencera à filtrer les messages entrants dès que vous aurez cliqué sur OK.

Pour gérer les filtres :

  1. Ouvrez une fenêtre Courrier.
  2. Ouvrez le menu &tasksMenu.label; et choisissez &filtersCmd.label;. La boîte de dialogue Filtres de messages s'affiche.
  3. Si vous avez plusieurs comptes de messagerie, sélectionnez celui dont vous désirez gérer les filtres.
  4. Sélectionnez parmi les options suivantes :
    Activer ou désactiver un filtre :
    Cochez la case à droite du nom du filtre pour activer le filtre et décochez-la pour le désactiver.
    Modifier un filtre :
    Sélectionnez le filtre et cliquez sur Édition (ou double-cliquez sur le nom du filtre). La boîte de dialogue Modifier le filtre s'ouvre pour vous permettre d'effectuer vos changements.
    Supprimer un filtre :
    Sélectionnez le nom du filtre et cliquez sur Effacer.
    Pour modifier l'ordre d'application des filtres :
    Dans la liste, cliquez sur le nom d'un filtre, puis sur Vers le haut ou Vers le bas pour le déplacer dans l'ordre d'application.

    Remarque : les filtres sont appliqués à chaque message entrant dans l'ordre de votre choix, jusqu'à ce qu'une action de filtre entraîne la suppression du message ou son déplacement dans un autre dossier.

  5. Cliquez sur OK lorsque vous avez terminé de gérer les filtres. Si vous avez créé un nouveau filtre, il commencera à filtrer les messages entrants dès que vous aurez cliqué sur OK.

Remarque : si vous supprimez un dossier utilisé pour stocker vos messages filtrés, le filtre ne fonctionnera plus. Les messages entrants correspondant aux critères de filtrage restent alors dans le dossier Courrier entrant. Si vous renommez ou déplacez un dossier utilisé pour stocker des messages filtrés, le filtre sera automatiquement mis à jour pour utiliser le nouveau dossier.

Astuce : les filtres ne s'appliquent pas aux messages existants. Si vous avez des messages que vous souhaitez déplacer vers un autre dossier, vous pouvez effectuer une recherche de ces messages puis les archiver dans le dossier en question.

Filtrer les messages d'un expéditeur spécifique

&brandShortName; vous permet de créer rapidement un filtre pour les messages d'un expéditeur donné. Par exemple, vous pouvez déplacer automatiquement tous les messages envoyés par le professeur de votre enfant dans un dossier nommé École.

Pour créer un filtre concernant un expéditeur spécifique :

  1. Ouvrez une fenêtre Courrier.
  2. Choisissez un message de cet expéditeur.
  3. Ouvrez le menu &msgsMenu.label;, et choisissez &createFilter.label;. La boîte de dialogue Modifier le filtre s'affiche. En utilisant l'adresse de l'expéditeur, &brandShortName; a configuré le nom, le critère de recherche et l'action du filtre (Déplacer vers un dossier).
  4. Choisissez un dossier de destination dans lequel archiver les messages entrants de l'expéditeur spécifié ou créez un nouveau dossier.
  5. Cliquez sur OK pour confirmer vos réglages. La boîte de dialogue Filtres de messages apparaît. Vous pouvez alors créer, supprimer ou modifier vos filtres de messages.
  6. Cliquez sur OK pour fermer la fenêtre des filtres de messages. Le filtre concernant les messages envoyés par l'expéditeur en question prendra effet dès que vous aurez cliqué sur OK.

Rechercher des messages

&brandShortName; Courrier & Groupes permet de rechercher rapidement du texte dans un message, de chercher des messages par objet, expéditeur ou d'utiliser une combinaison de critères pour soumettre tous les messages à une recherche approfondie dans un dossier, groupe ou compte de messagerie spécifique.

Pour rechercher du texte dans un seul message :

  1. Ouvrez une fenêtre Courrier.
  2. Sélectionnez le message, ouvrez le menu &editMenu.label; et choisissez &findCmd.label;.
  3. Entrez le texte à rechercher.
  4. Cliquez sur Rechercher pour trouver la première occurrence.
  5. Continuez à cliquer sur Rechercher pour trouver d'autres occurrences ou cliquez sur &undoDefaultCmd.label; une fois votre recherche terminée.
  6. Si vous avez fermé la fenêtre Recherche dans la page, sélectionnez &findAgainCmd.label; dans le menu &editMenu.label; pour continuer à rechercher le texte dans le reste du message.

Pour rechercher rapidement des messages par objet ou par expéditeur :

  1. Ouvrez une fenêtre Courrier.
  2. Dans le champ &searchNameAndEmail.placeholder;, entrez le texte du sujet ou le nom de l'expéditeur que vous désirez trouver. Vous pouvez entrer une partie du sujet ou de l'expéditeur ou entrer le nom exact que vous désirez trouver.

    Dès que vous arrêtez d'écrire, &brandShortName; Courrier & Groupes affiche les messages qui possèdent une occurrence ou un expéditeur contenant le texte recherché que vous êtes en train d'entrer.

  3. Une fois votre recherche terminée, cliquez sur Effacer pour supprimer le texte de recherche et afficher tous les messages.

Recherche précise de messages

Pour rechercher des messages dans des dossiers de messagerie ou de groupes :

  1. Ouvrez une fenêtre Courrier.
  2. Ouvrez le menu &tasksMenu.label; et choisissez &searchMailCmd.label;. La boîte de dialogue Rechercher les messages apparaît.
  3. Dans la liste déroulante Rechercher des messages dans, choisissez le compte, le groupe ou le dossier dans lequel vous souhaitez effectuer une recherche.
  4. Cochez Rechercher dans les sous-dossiers pour inclure tous les sous-dossiers dans la recherche.
  5. Sélectionnez Rechercher sur le système local pour rechercher uniquement les messages des groupes de discussion ou des comptes IMAP qui ont été enregistrés localement.

    Note : La case à cocher sera désactivée s'il n'est pas possible de faire une recherche sur un dossier de messages distant.

  6. Sélectionnez l'option correspondante pour rechercher des messages soit en respectant toutes les conditions soit d'après au moins une des conditions que vous choisirez.
  7. Utilisez la liste déroulante pour indiquer les critères de recherche (par exemple Objet et contient) puis entrez le texte ou la phrase que vous recherchez.

    Astuce : pour chercher des messages contenant un en-tête non listé dans la première liste déroulante (par exemple, si vous recherchez des messages qui incluent un champ "Resent-From"), choisissez Personnaliser et entrez l'en-tête dans lequel vous voulez chercher. &brandShortName; Courrier & Groupes ajoutera le nouvel en-tête à la liste déroulante. Ainsi, vous pourrez le choisir pour d'autres règles de messages. Assurez-vous d'avoir correctement saisi le nom du champ, car &brandShortName; Courrier & Groupes ne peut trouver que des correspondances exactes.

  8. Cliquez sur Plus pour ajouter des critères et sur Moins pour enlever le dernier critère de la liste.
  9. Cliquez sur Rechercher pour commencer la recherche ou sur Effacer pour réinitialiser les critères. Les résultats de la recherche apparaissent dans la zone Résultats de la boîte de dialogue Rechercher les messages.
    • Pour ouvrir un message et le lire, sélectionnez-le et cliquez sur Ouvrir ou double-cliquez dessus.
    • Pour trier les messages dans un ordre différent, cliquez sur la colonne à partir de laquelle trier.
    • Pour déplacer ou copier un message de la zone Résultats dans un autre dossier, sélectionnez-le et choisissez le dossier de destination dans la liste déroulante Classer. Si ce dossier est dans le même compte, le message y est déplacé. Si le dossier de destination appartient à un autre compte, il y est dupliqué.
    • Pour supprimer un message de la zone Résultats, sélectionnez le message puis cliquez sur Effacer.
    • Pour ouvrir le dossier dans lequel se trouve un message, sélectionnez-le et cliquez sur Aller au dossier du message.

Contrôler les courriers indésirables

&brandShortName; vous permet de filtrer les courriers que vous recevez et, si ceux-ci ne sont pas sollicités (également appelés spam en anglais) ou s'ils sont frauduleux (phishing en anglais), c'est-à-dire qu'ils reproduisent illégalement un site institutionnel afin de récupérer des données vous concernant), de les déplacer vers un dossier que vous aurez choisi puis de les effacer. Ce filtre est basé sur une analyse bayésienne, (une technique d'auto-apprentissage) et peut être utilisé pour jeter à la poubelle le courrier non sollicité. Cette fonction peut remplacer très efficacement le filtrage côté serveur ou être utilisé en concert avec celui-ci.

Utiliser le gestionnaire des indésirables

Le gestionnaire des indésirables vous permet de paramétrer les diverses actions à effectuer lorsque &brandShortName; rencontre un courrier supposé indésirable. Pour ouvrir le gestionnaire des indésirables, suivez ces étapes :

  1. Ouvrez une fenêtre Courrier & Groupes.
  2. Ouvrez le menu &tasksMenu.label; et sélectionnez Gestionnaire des indésirables.

Les messages supposés indésirables sont marqués par l'icône . Si vous ne voyez pas la colonne Statut indésirable, cliquez sur l'icône et sélectionnez Statut indésirable pour afficher cette colonne.

Le gestionnaire des indésirables vous offre la possibilité de paramétrer les options présentées ci-dessous. Chaque compte étant indépendant, si vous en possédez plusieurs il vous faudra les paramétrer séparément.

Utiliser le contrôle des messages indésirables

La fonctionnalité de contrôle des messages indésirables de &brandShortName; peut évaluer vos messages entrants, et repérer les possibles messages indésirables (ou non sollicités). La fonctionnalité utilise une classification bayésienne. Vous commencez par entrainer &brandShortName; en lui indiquant un ensemble de messages qui sont indésirables, et un ensemble de messages qui ne le sont pas. Ensuite, vous le laissez trier lui-même les nouveaux messages. Si &brandShortName; fait une erreur, vous pouvez la corriger.

Pour utiliser le contrôle des messages indésirables :

  1. Commencez par entraîner &brandShortName; à reconnaître les messages indésirables des autres messages. Il y a trois manières de changer le statut de désirabilité de(s) message(s) sélectionné(s) :
    • Ouvrez le menu &msgsMenu.label;, sélectionnez Marquer, et choisissez Comme indésirable ou Comme acceptable.
    • Cliquez sur le bouton Indésirable de la barre d'outils.

    • Cliquez pour modifier la colonne Statut indésirable dans la liste des messages. (Si vous ne le voyez pas, cliquez le bouton le plus à droite () dans la barre d'en-tête de la liste et sélectionnez Statut indésirable depuis la fenêtre surgissante.)

    Quand vous modifier le statut indésirable, une icône de poubelle apparaitra ou non dans la colonne correspondante, pour indiquer le statut indésirable du message sélectionné.

  2. Ouvrez le menu &editMenu.label;, et choisissez &accountManagerCmd.label;. Vous voyez la fenêtre &accountManagerTitle.label;.
  3. Cliquez sur la catégorie &junkSettings.label;, de votre compte courrier.
  4. Activez la fonctionnalité, et &brandShortName; classera automatiquement les messages arrivant (voir les options de contrôle des courriers indésirables. Des détails sur les autres paramètres peuvent être trouvée dans la description du panneau de préférences Paramètres pour les indésirables).
  5. Si vous l'avez entraîné sur des messages contenant des virus, pensez à désactiver la liste blanche (beaucoup de virus envoient des messages pour infecter les personnes contenues dans le carnet d'adresses des ordinateurs attaqués).
  6. Assurez vous de corriger les paramètres des courriers indésirables quand l'application fait une erreur de reconnaissance.
  7. Pour analyser les messages existants, sélectionnez les messages à analyser, ouvrez le menu Outils, et choisissez &runJunkControls.label;.

Note : &brandShortName; n'effectuera un contrôle des messages indésirables que lorsque la base de donnée d'entraînement a des informations sur les messages non indésirables. Si le contrôle ne fonctionne pas, sélectionnez quelques messages, et marquez les explicitement comme non indésirables.

Marquer les messages

Avant de pouvoir utiliser au mieux de ses capacités le gestionnaire des indésirables, il vous faudra marquer manuellement les messages que vous considérez comme indésirables afin de lui apprendre à les reconnaître. Pour cela, un échantillon d'une trentaine de messages est nécessaire. Plus vous marquerez de messages comme acceptables et/ou indésirables, plus le gestionnaire des indésirables sera efficace.

Marquer un message comme indésirable

Pour marquer un message comme indésirable, suivez les étapes suivantes :

  1. Sélectionnez un message que vous voulez marquer comme indésirable.
  2. Puis exécutez l'une des actions suivantes :
    • Ouvrez le menu &msgsMenu.label;, sélectionnez Marquer puis choisissez Comme indésirable.
    • Cliquez sur le bouton Indésirable de la barre d'outils. Remarquez que ce bouton change de dénomination une fois cliqué pour devenir Acceptable.
    • Dans la colonne Statut indésirable, cliquez sur le point pour marquer le message. L'icône apparaît dans cette colonne en face du message marqué comme indésirable
  3. Ouvrez le menu &tasksMenu.label; et choisissez Gestionnaire des indésirables…. Activez le gestionnaire des indésirables pour votre compte de messagerie, et &brandShortName; classera automatiquement les messages entrants. Pour plus d'informations, voir la section Gestionnaire des indésirables.
  4. De nombreux virus de courriel utilisent les carnets d'adresses des clients de messagerie des personnes infectées. Si vous avez entraîné le gestionnaire des indésirables sur des messages infectés par des vers ou des virus, il est fortement conseillé de désactiver l'option Ne pas marquer les messages comme indésirables si l'expéditeur est dans mon carnet d'adresses.
  5. Au cas où le gestionnaire des indésirables commettrait une erreur dans la classification d'un courriel, corrigez-la manuellement, soit en utilisant l'icône Indésirable/Acceptable, soit en utilisant le menu &msgsMenu.label;.
  6. Pour analyser les messages d'un dossier, sélectionnez &runJunkControls.label;.
  7. Pour effacer directement et simplement les messages indésirables d'un dossier, ouvrez le menu &tasksMenu.label; et sélectionnez &deleteJunk.label;.

Les messages supposés indésirables sont marqués par l'icône . Si vous ne voyez pas la colonne Statut indésirable, cliquez sur l'icône et sélectionnez Statut indésirable pour afficher cette colonne.

Marquer un message comme acceptable

En plus des actions similaires aux précédentes, vous disposez d'un moyen supplémentaire pour marquer un message comme acceptable. Lorsqu'un message est marqué comme indésirable, une barre d'outils apparaît au dessus de l'en-tête du message avec le texte suivant : &junkBarMessage.label;. Si vous estimez que ce message n'est pas indésirable, appuyez sur le bouton Acceptable.

Marquer les messages

Pour configurer finement le contrôle des messages indésirables, utilisez les préférences globales des courriers indésirables et suspects pour les paramètres non dépendants des comptes et les Paramètres des indésirables du gestionnaires de comptes pour les paramètres spécifiques à un compte courrier.

Gestionnaire des indésirables et filtres des messages

Le gestionnaire des indésirables s'exécute après les filtres que vous avez créés et ne s'applique que sur les boîtes de réception et leurs sous-dossiers. Vous pouvez utiliser cette particularité à votre avantage. Par exemple, vous pouvez filtrer les messages dont vous êtes sûr en les déplaçant hors de votre boîte de réception pour éviter qu'ils ne soient évalués par le gestionnaire des indésirables (particulièrement utile pour les lettres d'information et les listes de diffusion).

Filtres et contrôle des indésirables

Le contrôle des indérisables n'est lancé qu'après les filtres courriers, et ne s'applique qu'au dossier Courrier entrant et à ses sous-dossiers. Utilisez le à votre avantage, par exemple, vous pouvez filtrer les courriers dont vous êtes sûr qu'ils ne sont pas indésirables dans un dossier spécial, en dehors du dossier Courrier entrant, de telle manière que les messages ne seront pas classés comme indésirables (très utile si vous vous inscrivez à des lettres d'informations ou que vous êtes sur une liste de diffusion modérée).

Détection de l'hameçonnage

L'hameçonnage (phishing en anglais) est une technique courante d'escroquerie, où quelqu'un créé un faux site, destiné à piéger ses destinataires, et à ce qu'ils lui transmettent des informations personnelles comme des numéros de carte de crédit, des nom d'utilisateur et mots de passe, ou encore des numéros de sécurité sociale. En détournant les noms de grandes banques, de vendeurs en ligne et de sociétés de carte de crédit, les hameçonneurs convainquent souvent les destinataires de donner suite à leurs avances.

Dans de nombreux cas, vous recevrez un lien vers une page d'hameçonnage, par un courrier qui se réclame comme émanant d'une origine officielle. Il peut arriver que vous tombiez sur ce type de pages en suivant des liens sur internet, ou par messagerie instantanée.

Astuce : Étant donné qu'une URL forgée peut être très ressemblante à une officielle, il est plus sûr d'utiliser un marque-page que vous avez créé, ou de taper l'URL dans la barre d'adresse, à la main, au lieu de suivre un lien dans un courrier. Pensez toujours au risque d'une URL forgée s'il vous est demandé de vous authentifié ou de fournir des informations personnelles sur un site web.

Le détecteur de tentatives d'hameçonnage de &brandShortName; est activé par défaut. Quand il rencontre un courrier qui semble être frauduleux, une barre d'avertissement s'affichera dans la fenêtre du message.

Si vous pensez que ce courrier n'est pas frauduleux, cliquez sur le bouton &removePhishingBarButton.label; button, et la barre d'avertissement disparaitra.

Quand un utilisateur clique sur un lien dans un courrier qui semble être une URL d'hameçonnage, &brandShortName; affichera une boîte de dialogue à l'utilisateur avant que le site internet ne soit ouvert.

Cette fenêtre apparaitra si l'une des trois conditions suivantes est remplie : le nom de domaine de l'URL est une addresse IP, ou le texte du lien est une URL, dont le nom de domaine ne correspond pas à celui de l'URL réelle.

Note : La détection de l'hameçonnage a une priorité supérieure à celle de la détection des courriers indésirables.

Pour plus d'informations techniques sur le sujet, voyez le document Know your Enemy : Phishing.
