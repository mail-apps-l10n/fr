Utiliser l'assistant de configuration du courrier

Pour configurer un compte de courrier ou de groupes, ouvrez d'abord le menu &windowMenu.label; et choisissez Courrier. Si vous n'avez pas encore configuré de compte, l'assistant de création de compte s'ouvre automatiquement, pour vous permettre de le faire.

Cet assistant vous guide dans le processus de création d'un nouveau compte. Si vous ne connaissez pas un paramètre, cliquez sur &undoDefaultCmd.label; et demandez à votre fournisseur d'accès à internet (FAI) ou à votre service d'assistance.

Si un compte est déjà configuré, l'assistant de création de compte n'apparaîtra pas. Dans ce cas, après avoir ouvert la fenêtre Courrier, ouvrez le menu &fileMenu.label; et choisissez &newMenu.label;, puis &newAccountCmd.label;. Pour plus d'informations, voir la section Configurer des comptes de courrier ou de groupes supplémentaires.

Configurer des comptes de messagerie d'un FAI ou d'un fournisseur d'adresse électronique

Avant de configurer un compte de messagerie, votre FAI ou votre fournisseur d'adresse électronique devra vous fournir les informations suivantes :

  • votre nom d'utilisateur ;
  • votre adresse électronique ;
  • les noms des serveurs de messagerie entrant et sortant (par exemple : pop.laposte.net et smtp.laposte.net) ;
  • le type de serveur de messagerie entrant (IMAP ou POP).

Pour configurer un compte de groupes, votre FAI ou votre fournisseur d'adresse électronique devra vous fournir les informations suivantes :

  • votre adresse électronique ;
  • le nom du serveur de groupes ;
  • le nom du compte.

Pour configurer un nouveau compte de messagerie ou de groupes, ouvrez la fenêtre Courrier :

  1. Ouvrez le menu SeaMonkey &editMenu.label; et choisissez &accountManagerCmd.label;. La fenêtre Paramètres des comptes courrier et groupes s'ouvre.
  2. Cliquez sur &addAccountButton.label; pour démarrer l'assistant de création de compte.

    Ce que l'assistant vous demande de fournir dépend du type de compte que vous allez créer. Dans les lignes qui suivent, le titre en gras correspond à la fenêtre que vous verrez lorsque vous configurerez un compte de messagerie de votre FAI ou d'un fournisseur d'adresses électroniques.

  3. Remplissez les différents champs comme indiqué ci-dessous :
    &accountTypeTitle.label;
    Choisissez le type de compte que vous voulez, puis cliquez sur Suivant.
    &identityTitle.label;
    Entrez le nom et l'adresse électronique correspondants à ce compte, puis cliquez sur Suivant.
    &serverTitle.label;
    précisez si vous voulez un compte POP ou IMAP. Tous les prestataires ne fournissent pas les deux options. Pour plus d'informations, voir la section &accountManagerTitle.label; - Paramètres serveur.

    Vous devez aussi fournir ici les noms de vos serveurs de messagerie entrant et sortant (SMTP). Il s'agit du nom du serveur de messagerie qui expédie vos messages (serveur SMTP). Cliquez sur Suivant pour continuer.

    Remarque : vous n'avez besoin de fournir qu'un seul serveur de messagerie sortant SMTP, même si vous avez plusieurs comptes de messagerie. Vous n'avez pas nécessairement fourni le nom de votre serveur SMTP de manière explicite lors de la configuration du compte. Par exemple, votre serveur SMTP peut-être le même que votre serveur POP ou IMAP. En cas de doute, contactez votre FAI ou votre administrateur système.

    &fullnameLabel.label;
    saisissez le nom par lequel vous voulez appeler ce compte, puis cliquez sur Suivant.
    &emailLabel.label;
    saisissez le nom d'utilisateur fourni par votre FAI ou votre fournisseur d'adresse électronique, puis cliquez sur Suivant.
  4. &completionTitle.label; Vérifiez les informations que vous avez entrées avec votre FAI ou administrateur système. Lorsque vous êtes sûr qu'elles sont correctes, cliquez sur Terminer pour créer votre compte.
  5. Vous pouvez voir votre nouveau compte dans la liste à gauche de la boîte de dialogue &accountManagerTitle.label;. Cliquez sur OK pour commencer à utiliser votre nouveau compte.

Vous êtes maintenant prêt à récupérer les messages de votre compte. &brandShortName; Courrier et groupes vous demandera votre mot de passe quand vous récupérez votre courrier à la première connexion de chaque session. Pour des instructions détaillées, voir la section Recevoir de nouveaux messages.

Configurer des comptes de messagerie ou groupes supplémentaires

Pour changer les paramètres d'un compte ou en ajouter un nouveau, utilisez la fenêtre &accountManagerTitle.label;. Vous pouvez alors changer :

  • les paramètres des serveurs de courrier et groupes (par exemple, les paramètres de suppression ou de téléchargement des messages) ;
  • les paramètres de stockage des copies de messages et des dossiers ;
  • l'adresse à laquelle vous voulez qu'on réponde, le nom de votre organisation et votre signature.

Pour ajouter un nouveau compte ou changer les paramètres d'un compte existant, à partir de la fenêtre Courrier :

  1. Choisissez &accountManagerCmd.label; dans le menu &editMenu.label;. La fenêtre &accountManagerTitle.label; s'ouvre. Vous pouvez alors effectuer ces actions :
    &addAccountButton.label;
    Cliquez sur ce bouton pour configurer un nouveau compte de messagerie ou de groupes. Assurez-vous de spécifier les informations du compte exactement comme on vous les a données. Cliquez sur Suivant et Précédent pour vous déplacer entre les écrans. Pour annuler la création du compte, cliquez sur &undoDefaultCmd.label;.
    &setDefaultButton.label; :
    Sélectionnez un compte et cliquez sur ce bouton pour faire apparaître le compte sélectionné en tête de la liste des comptes dans la fenêtre Courrier. Les changements ne prendront effet qu'une fois que vous aurez redémarré &brandShortName; Courrier & Groupes.

    Le compte par défaut est celui auquel vous voulez vous connecter et (pour les comptes IMAP seulement) vérifier automatiquement quand vous démarrez &brandShortName; Courrier & Groupes. Pour les comptes POP, vous devez toujours cliquer sur le bouton Relever pour recevoir les nouveaux messages.

    &removeButton.label; :
    Sélectionnez un compte, puis cliquez sur ce bouton pour le supprimer totalement.
    &smtpServer.label;
    Sélectionnez &smtpServer.label; (au bas de la liste des comptes) pour modifier des informations sur le serveur de courrier sortant. Pour plus d'informations, voir la section Paramètres des comptes - Serveur sortant (SMTP).
  2. Cliquez sur les éléments de la liste sous les noms des comptes pour modifier les paramètres correspondants sur le panneau de droite.
  3. Cliquez sur OK pour enregistrer vos changements.

Modifier les paramètres d'un compte

Pour voir ou modifier les informations d'un compte courrier ou groupes existant, ouvrez la fenêtre Courrier :

  1. Ouvrez le menu &editMenu.label; et choisissez &accountManagerCmd.label;. La fenêtre &accountManagerTitle.label; s'ouvre.
  2. Cliquez sur le nom du compte dans la partie gauche de la fenêtre. Les paramètres des comptes, comme votre adresse électronique ou votre signature s'affichent dans la partie droite de la fenêtre.
  3. Cliquez sur l'une des entrées sous le nom d'un compte pour voir les paramètres correspondants :
    Paramètres serveur :
    Les paramètres accessibles dépendent du type de serveur (IMAP, POP, ou serveur de groupes). Pour plus d'informations, voir la section Paramètres des comptes courrier & groupes - Paramètres serveur.

    Important : si vous avez besoin de changer le type de votre serveur (par exemple, passer de POP à IMAP) vous devez commencer par supprimer le compte existant. Quittez ensuite &brandShortName; et redémarrez-le. Vous pouvez alors rouvrir la fenêtre &accountManagerTitle.label; et recréer le compte avec un nouveau type de serveur en cliquant sur &addAccountButton.label;.

    Copies et Dossiers :
    Ces paramètres déterminent s'il faut envoyer des messages automatiques (copie carbone invisible) et le dossier où vous souhaitez conserver des copies des messages sortants, des brouillons et des modèles de messages. Pour plus d'informations, voir la section Paramètres des comptes courrier & groupes - Copies et dossiers.
    &addressing.label;
    Ces paramètres vous permettent de définir le format d'envoi d'un nouveau message ainsi que les préférences d'affichage des citations. Vous pouvez également choisir d'ignorer les paramètres globaux de serveur d'annuaire spécifiés pour tous les carnets d'adresses dans les préférences. Pour plus d'informations, voir la section Paramètres des comptes courrier & groupes - &addressing.label;.
    Hors ligne & espace disque (comptes IMAP et de groupes seulement) :
    Ces paramètres s'appliquent lorsque vous êtes hors ligne (déconnecté d'Internet) ou que vous avez besoin d'économiser du temps de téléchargement et de l'espace disque. Pour plus d'informations, voir la section Hors ligne et espace disque (IMAP) ou Hors ligne et espace disque (Groupes).
    &diskspaceGroupTitle.label; (comptes POP seuls) :
    Ce paramètre détermine la taille maximale des messages que vous voulez télécharger sur votre disque dur. Pour plus d'informations, voir la section Espace disque (POP).
    &pref.receipts.title;
    Ces paramètres déterminent le comportement de &brandShortName; lors de la réception d'un message demandant un accusé de réception ou lors de l'envoi d'un message, s'il doit en réclamer un.
    Sécurité :
    Ces paramètres déterminent quels certificats sont utilisés pour signer numériquement et chiffrer les messages que vous expédiez. Les signatures numériques permettent à vos correspondants d'être sûr que vous êtes bien l'auteur du message. Le chiffrement permet d'être sûr que vos messages resteront privés durant leur transit. Pour plus d'informations, voir la section Paramètres des comptes courrier & groupes - Sécurité.
  4. Cliquez sur OK pour valider vos modifications.

Recevoir de nouveaux messages

Pour un compte IMAP, vous pouvez les collecter automatiquement et les afficher dans le dossier Courrier entrant en lançant &brandShortName; Courrier & Groupes et en sélectionnant le Courrier entrant d'un compte IMAP.

Pour un compte POP, il vous faut sélectionner le dossier Courrier entrant et cliquer sur Relever pour collecter vos messages. Par défaut, quand vous collectez vos messages depuis votre compte POP, les messages sont télécharger en entier puis effacés du serveur POP. Vous avez la possibilité de changer les paramètres de votre serveur POP pour ne récupérer que les en-têtes et/ou laisser une copie des messages sur le serveur.

Vous pouvez aussi configurer &brandShortName; Courrier & Groupes pour qu'il collecte les nouveaux messages au démarrage et qu'il vérifie à intervalles réguliers s'il n'y a pas de nouveaux messages.

Icône de Courrier & Groupes

Quand de nouveaux messages sont arrivés, l'icône dans la barre d'état affiche une flèche verte.

Notification de nouveaux messages

Pour configurer &brandShortName; Courrier & Groupes afin qu'il vérifie automatiquement s'il y a des nouveaux messages, ouvrez la fenêtre Courrier :

  1. Ouvrez le menu &editMenu.label; et choisissez &accountManagerCmd.label;. La fenêtre &accountManagerTitle.label; s'affiche.
  2. Si vous avez plusieurs comptes, sélectionnez-en un et cliquez sur la catégorie Paramètres serveur correspondante.
  3. Choisissez maintenant une des deux options suivantes :
    &loginAtStartup.label; :
    Sélectionnez cette case si vous voulez vérifier les nouveau messages de ce compte automatiquement lorsque Courrier & Groupes démarre. Pour les comptes POP, &brandShortName; vérifie la présence de nouveaux messages, mais ne les télécharge pas tant que vous n'avez pas cliqué sur Relever ou choisi &downloadOnBiff.label;.
    &biffStart.label; [__] &biffEnd.label; :
    Sélectionnez cette case si vous voulez définir un intervalle de temps entre deux vérifications. Vous pouvez aussi vérifier à tout moment si vous n'avez pas de nouveau message en cliquant sur Relever.
  4. Cliquez sur OK. Vos paramètres s'appliqueront au prochain lancement de &brandShortName; Courrier & Groupes.

Pour demander à &brandShortName; Courrier & Groupes de jouer un son ou d'afficher un avertissement lorsque vous recevez un nouveau message, voir la section Définition des préférences générales de Courrier & Groupes.

Vous pouvez de toute façon collecter vos nouveaux messages à tout moment. Pour recevoir les messages du compte courrier ou groupes actuellement sélectionné, vous pouvez :

  • Cliquer sur Relever dans la barre d'outils.

    ou

  • Ouvrir le menu &fileMenu.label; et choisir Relever les nouveaux messages.

Pour recevoir les nouveaux messages de tous les comptes, ouvrez la fenêtre Courrier :

  1. Cliquez sur le triangle à côté du bouton Relever dans la barre d'outils.
  2. Choisissez &getAllNewMsgCmd.label;. &brandShortName; Courrier & Groupes collecte alors les nouveaux messages de tous les comptes de messageries.

    Si vous n'êtes pas déjà connecté à un compte courrier, &brandShortName; Courrier & Groupes vous demandera d'abord de saisir votre nom d'utilisateur et votre mot de passe avant de collecter les messages de ce compte. Si votre nom d'utilisateur et votre mot de passe ont été mémorisés par le gestionnaire de mots de passe, &brandShortName; Courrier & Groupes ne vous demandera pas cette information.

Remarque : vous pouvez également ouvrir le menu &fileMenu.label; (dans la fenêtre Courrier) et choisir &getNewMsgForCmd.label;.

Pour recevoir les nouveaux messages d'un compte courrier spécifique, ouvrez la fenêtre Courrier :

  1. Cliquez sur le triangle à côté de Relever dans la barre d'outils.
  2. Choisissez le compte pour lequel vous voulez collecter les messages.

Remarque : &brandShortName; Courrier & Groupes vous demande votre mot de passe la première fois que vous collectez vos messages pour un compte donné. Vous pouvez à ce moment-là choisir si &brandShortName; Courrier & Groupes peut mémoriser votre mot de passe dans le gestionnaire de mots de passe.

Le gestionnaire de mots de passe peut enregistrer tous vos noms d'utilisateur et mots de passe. Pour plus d'informations, voir la section Utiliser le gestionnaire de mots de passe.

Classer et ordonner les messages

Pour ordonner les messages par catégorie (sujet, expéditeur, date ou priorité), ouvrez la fenêtre Courrier :

  • Cliquez sur la colonne correspondant au type de tri que vous voulez faire dans la liste des messages. Vous pouvez également ouvrir le menu &viewMenu.label;, choisir &sortMenu.label;, puis sélectionnez la colonne correspondant au type de tri que vous désirez obtenir.

Pour réorganiser l'ordre des colonnes, ouvrez la fenêtre Courrier :

  • Cliquez et déplacez le titre de la colonne à gauche ou à droite pour la déplacer.

Pour changer les colonnes affichées, rendez-vous dans la fenêtre Courrier :

  • Cliquez sur l'icône Choisir les colonnes à afficher et sélectionnez la colonne à ajouter ou enlever de la liste.

Pour regrouper les messages par sujet, de manière à ce que chaque message soit suivi de ses réponses :

  • Cliquez sur le bouton à la gauche des colonnes Sujet, Expéditeur et Date. Vous pouvez également ouvrir le menu &viewMenu.label;, choisir &sortMenu.label;, puis sélectionner &sortBySubjectCmd.label;.
Bouton de tri par fil de discussion

Astuces :

  • Pour vous aider à identifier les messages non lus lorsque des messages dans un même fil ont été précédemment lus, &brandShortName; Courrier & Groupes souligne le message ouvrant le fil de discussion.
  • Le bouton Discussion ordonne les fils automatiquement d'après l'âge des messages. Si vous désirez utiliser un autre critère de classement, ouvrez le menu &viewMenu.label; et sélectionnez l'option voulue à partir du sous-menu Trier par.
  • Sélectionnez &preserveThreading.label; dans les Préférences de courrier et groupes de discussion si vous voulez que &brandShortName; conserve le fil des discussions quand vous cliquez sur les titres de colonnes pour trier vos messages. Le bouton des discussions bascule entre affichage groupé par fil et non groupé dans ce mode. Si &preserveThreading.label; n'est pas sélectionné, &brandShortName; affiche automatiquement les messages non groupés par fil quand vous triez en cliquant sur les titres de colonnes.

Enregistrer et imprimer des messages

Pour enregistrer un message au format texte, HTML ou Outlook Express :

  1. Sélectionnez le message dans la fenêtre Courrier.
  2. Ouvrez le menu &fileMenu.label;, choisissez &saveAsMenu.label; puis &saveAsFileCmd.label;.
  3. Choisissez un type de fichier (HTML, texte ou fichier de courrier). Choisissez fichier de courrier si vous souhaitez pouvoir ouvrir le message avec Microsoft Outlook ou Microsoft Outlook Express.
  4. Modifiez l'extension du nom de fichier pour qu'il se termine en .html, .txt ou .eml, selon le type de fichier qui vous avez choisi à l'étape précédente.
  5. Choisissez un emplacement, puis cliquez sur Enregistrer.

Pour imprimer un message sélectionné :

  • Cliquez sur Imprimer.

Contrôler les images, scripts et plugins

Par défaut, les images et autres pièces jointes distantes ne sont pas visibles dans les messages que vous recevez, à l'exception des messages provenant de personnes autorisées présentes dans votre carnet d'adresses personnel. Pour modifier les paramètres d'affichage des pièces jointes, suivez ces étapes :

  1. Ouvrez le menu SeaMonkey&editMenu.label; et choisissez &preferencesCmd.label;.
  2. Sous la catégorie Confidentialité et Sécurité, cliquez sur Images. Si aucune sous-catégorie n'est visible, double-cliquez sur Confidentialité et Sécurité pour développer la liste.
  3. Dans la section Générale, vous pouvez faire :
    Bloquer les images et autres contenus depuis les sources distantes
    Décocher cette case pour autoriser tout le contenu distant.
  4. Cliquez sur OK pour valider vos changements.

Note : Voir &allowRemoteContent1.label; Créer une fiche de carnet d'adresses pour plus de détails sur la manière de définir quels expéditeurs seront autorisés à afficher du contenu distant.

Par défaut, JavaScript et les plugins ne sont pas activés pour la messagerie. Pour modifier ces paramètres :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et choisissez &preferencesCmd.label;.
  2. Sous la catégorie Avancées, cliquez sur Scripts et plugins. Si aucune sous-catégories n'est visible, double-cliquez sur Avancées pour développer la liste.
  3. Sous Activer JavaScript pour, cochez le Courrier et les Groupes pour activer JavaScript dans les courriels.
  4. Sous Activer les plugins pour, cochez le Courrier et les groupes pour désactiver les plugins.
  5. Cliquez sur OK pour valider vos changements

Écrire des messages pour le courrier ou les groupes

Voici comment choisir le destinataire, écrire ou répondre à un message ou en envoyer un nouveau :

  • Dans n'importe quelle fenêtre de &brandShortName;, ouvrez le menu &fileMenu.label; et faites &newMenu.label; puis &newNewMsgCmd.label;. Vous pouvez aussi cliquer directement sur Nouveau dans la barre d'outils de &brandShortName; Courrier & Groupes.
  • Quand vous visualisez un message, cliquez sur &replyMsgCmd.label;, &forwardMsgCmd.label; ou &replyToAllMsgCmd.label; dans la barre d'outils Courrier.
  • Dans le carnet d'adresses, sélectionnez une adresse puis cliquez sur Rédiger.

Astuce : dans la boîte de dialogue &accountManagerTitle.label; &addressing.label; du menu &editMenu.label;, vous pouvez spécifier quel éditeur (texte ou HTML) sera utilisé pour la rédaction des messages pour ce compte (vous pouvez spécifier un éditeur différent pour chaque compte). Sélectionnez un compte et cochez la case Composer les messages en HTML afin d'utiliser l'éditeur HTML pour rédiger vos messages. Pour plus d'informations, voir la section Changer les paramètres d'un compte.

Écrire des messages au format HTML permet d'utiliser dans vos messages différentes polices, styles de texte (tels que gras ou italique), couleurs de texte, tableaux, listes à puces ou numérotées, et des images. Toutefois, il est possible que certains destinataires ne puissent lire que les messages en texte simple. Si vous souhaitez utiliser un éditeur de texte de temps en temps, vous pouvez presser la touche Maj en cliquant le bouton Nouveau ou &replyMsgCmd.label;.

Commencez d'abord par spécifier si vous voulez écrire vos messages en texte seulement ou en HTML. Allez pour cela dans les Paramètres des comptes courrier & groupes - &addressing.label; :

  • Choisissez &accountManagerCmd.label;.

Pour voir la fenêtre &compositionGroupTitle.label;, cliquez sur Nouveau dans la barre d'outils de la fenêtre Courrier.

La fenêtre de rédaction des messages est organisée comme suit :

  • La barre d'outils

    Vérifie l'orthographe du texte du message.

À propos des carnets d'adresses

Les carnets d'adresses contiennent des informations (coordonnées professionnelles et/ou personnelles, nom, prénom, moyens de contact, etc.) concernant toute personne ou organisme pour lequel vous créerez une fiche. &brandShortName; Courrier & Groupes fournit deux carnets d'adresses : le carnet Adresses personnelles et le carnet Adresses collectées. Vous pouvez en créer d'autres (amis, famille, collègues, etc.). Vous pouvez aussi importer des carnets d'adresses à partir d'autres logiciels de messagerie électronique ou de versions précédentes de &brandShortName;. Le contenu de ces carnets est enregistré localement sur le disque dur, dans les dossiers de votre profil.

Le carnet d'adresses peut également contenir des adresses électroniques d'un annuaire LDAP, situé sur un serveur d'annuaire LDAP. Le serveur d'annuaire enregistre les adresses électroniques de personnes non incluses dans vos carnets d'adresses locaux. Le protocole LDAP est une méthode standard pour accéder à des services d'annuaire sur Internet ou sur un intranet, un annuaire d'entreprise par exemple.

Adresses personnelles

Utilisez le carnet d'adresses personnelles pour ajouter des noms spécifiques de votre choix. Vous pouvez créer des listes de diffusion et modifier des entrées du carnet.

Adresses collectées

Par défaut, le carnet d'adresses collectées récupère automatiquement les adresses électroniques contenues dans les messages que vous envoyez. Ces adresses sont enregistrées au moment où vous cliquez sur Envoyer.

Pour modifier ces paramètres, ouvrez une fenêtre Courrier puis :

    • Dans la liste déroulante, sélectionnez le carnet d'adresses où elles seront stockées.
  4. Cliquez sur OK.

Ajouter et retirer des annuaires LDAP

Ajouter un annuaire LDAP à votre carnet d'adresses vous permet de faire des recherches d'adresses électroniques et d'autres informations dans un annuaire distant, situé sur Internet ou sur un intranet. Vous pouvez aussi utiliser l'annuaire avec l'auto-complétion d'adresses de vos messages.

En général, vous ajoutez ou enlevez des serveurs d'annuaires en suivant les instructions fournies par votre administrateur système. Informez-vous auprès de lui pour connaître la procédure d'ajout d'un nouveau serveur d'annuaires.

Pour ajouter un nouvel annuaire :

  1. Ouvrez le menu &fileMenu.label;, choisissez &newMenu.label; puis Annuaire LDAP. La boîte de dialogue &newDirectoryTitle.label; apparaît.
  2. Saisissez les informations suivantes dans la boîte de dialogue &newDirectoryTitle.label; :
    &directoryName.label;
    entrez le nom du service d'annuaire (par exemple, Répertoire InfoSpace).
    &directoryHostname.label;
    Entrez l'adresse du serveur (par exemple, ldap.infospace.com).
    &directoryBaseDN.label;
    Ce paramètre définit le nom distinct de base à être utilisé. Pour limiter la recherche à un pays spécifique ou à une organisation, entrez par exemple c=JP, qui limite la recherche au Japon, et/ou o=Mozilla Communications Corporation pour vous limiter à cet organisme.
    &portNumber.label;
    Entrez le numéro de port du serveur LDAP. Par défaut c'est le 389.
    &directoryLogin.label;
    Le nom distinct utilisé pour s'authentifier (se connecter) au serveur LDAP. S'il est laissé en blanc, la connexion sera anonyme.
    &directorySecure.label;
    Cochez cette case si le serveur LDAP supporte les connexions sécurisées (chiffrées). Si vous n'êtes pas sûr, contactez l'administrateur système du serveur.
  3. Cliquez sur l'onglet &Advanced.tab; pour configurer le serveur d'annuaire LDAP.
  4. Entrez les informations suivantes :
    &return.label; [_] &results.label;
    Ce choix limite le nombre d'occurrences de l'auto-complétion retournées par le serveur d'annuaires.
    &scope.label;
    Définit les limites de la recherche. Choisissez :
    • &scopeOneLevel.label; pour trouver les entrées correspondantes en recherchant dans le nom distinct de la base et le niveau juste au-dessous.
    • &scopeSubtree.label; pour trouver les entrées correspondantes en recherchant dans le nom distinct de la base et tous les niveaux au-dessous. C'est la recherche la moins restrictive.
    &searchFilter.label;
    Filtre à appliquer aux résultats valides qui sont dans l'étendue de la recherche.
  5. Cliquez sur OK pour fermer la boîte de dialogue &newDirectoryTitle.label;.

L'annuaire que vous venez d'ajouter apparaît dans la liste des carnets d'adresses dans la fenêtre Carnet d'adresses.

Partager un dossier avec d'autres utilisateurs (IMAP seulement)

Les utilisateurs avec des comptes courrier IMAP peuvent partager des dossiers avec d'autres utilisateurs du même réseau. Partager des dossiers permet à plusieurs utilisateurs de voir et de travailler sur les mêmes messages, comme dans un groupe. Pour utiliser le partage de dossiers, votre serveur IMAP doit supporter la gestion des listes de contrôle d'accès (ACL, Access Control List). Demandez à votre administrateur système si vous n'êtes pas sûr que les dossiers partagés sont supportés par votre serveur IMAP.

Pour partager un dossier avec d'autres utilisateurs de votre réseau ou pour voir des informations partagées :

  1. Ouvrez une fenêtre Courrier.
  2. Dans un compte IMAP, sélectionnez un dossier que vous voulez partager ou un dossier dont vous voulez voir les privilèges de partage.

    Remarque : les dossiers dans les dossiers locaux ou dans un compte POP ne peuvent être partagés.

  3. Ouvrez le menu &editMenu.label; et choisissez &folderPropsFolderCmd.label;.
  4. Cliquez sur l'onglet Partage.
  5. Cliquez sur Privilèges. Vous pouvez avoir à saisir vos nom d'utilisateur et mot de passe.

    Le bouton Privilèges n'est disponible que si le serveur IMAP vous permet de modifier les privilèges de partage des dossiers. Si ce bouton n'est pas disponible, vous pouvez voir les privilèges de partage de ce dossier, mais pas les modifier.

  6. Suivez les instructions sur l'écran pour ajouter des utilisateurs et définir leurs droits d'accès aux dossiers.
    Privilèges de lecture :
    Les utilisateurs peuvent lire les messages et copier leur contenu, mais ne peuvent ni les modifier, ni les effacer, ni copier de nouveaux messages dans le dossier. Les utilisateurs peuvent marquer les messages comme lus ou non lus. Voir la section Étiqueter ou marquer des messages pour des détails sur le marquage des messages.
    Privilèges de lecture-écriture :
    En plus des privilèges de lecture, les utilisateurs peuvent modifier et supprimer des messages. Les utilisateurs peuvent aussi copier ou déplacer des messages dans le dossier.
    Privilèges d'administration :
    En plus des privilèges de lecture-écriture, les utilisateurs peuvent ajouter et supprimer des utilisateurs et changer leurs privilèges sur les dossiers.
  7. Cliquez sur OK pour confirmer vos modifications.
  8. Cliquez sur OK pour fermer la boîte de dialogue Propriétés du dossier.

Dans la liste des dossiers de votre compte de courrier, un dossier partagé se distingue par son icône.

Pour envoyer un message d'instructions d'accès à votre dossier partagé à partir d'une fenêtre Courrier :

    Seuls les destinataires sur le même réseau pourront accéder à votre dossier partagé.

  6. Cliquez sur Envoyer.

Créer des filtres de messages

Les filtres de messages permettent de gérer et d'organiser les messages automatiquement. Vous pouvez créer des filtres utilisés par &brandShortName; Courrier & Groupes pour effectuer automatiquement certaines actions sur les messages entrants en fonction de critères spécifiques. Par exemple, vous pouvez créer un filtre de messages qui déplacera automatiquement les messages entrants dans un dossier ou sous-dossier. Les filtres de messages sont conçus pour un compte particulier.

Pour créer un filtre de messages :

  1. Ouvrez une fenêtre Courrier.
  2. Ouvrez le menu &tasksMenu.label; et choisissez &filtersCmd.label;. La boîte de dialogue Filtres de messages s'affiche.
  3. Si vous avez plusieurs comptes de messagerie, sélectionnez celui sur lequel vous souhaitez appliquer le filtre.
  4. Cliquez sur Nouveau. La boîte de dialogue Modifier le filtre permet de spécifier les types de messages concernés et l'action à effectuer par le filtre.
  5. Entrez un nom pour le filtre.
  6. Sélectionnez l'option de condition qui convient au filtre que vous souhaitez appliquer : valident TOUTES les conditions suivantes ou valident au moins UNE des conditions suivantes.
  7. Utilisez les listes déroulantes pour sélectionner les critères de recherche (par exemple, Objet, Expéditeur, Contient, Ne contient pas) et entrez le texte à rechercher.

    Astuce : pour chercher des messages contenant un en-tête non listé dans la première liste déroulante (par exemple, si vous recherchez des messages qui incluent un champ Resent-From), choisissez Personnaliser et entrez l'en-tête dans laquelle vous voulez chercher. &brandShortName; Courrier & Groupes ajoutera le nouvel en-tête à la liste déroulante. Ainsi, vous pourrez le choisir pour d'autres règles de messages. Assurez-vous d'avoir correctement saisi le nom du champ, car &brandShortName; Courrier & Groupes ne peut trouver que des correspondances exactes.

  8. Cliquez sur Plus pour ajouter des critères et sur Moins pour supprimer le dernier critère de la liste.
  9. Cochez les cases pour sélectionner l'action (ou les actions) que vous souhaitez que le filtre applique aux messages (par exemple Déplacer vers le dossier). Si elles existent, utilisez les listes déroulantes en face de chaque option, afin de préciser l'action à exécuter.

    Astuce : pour étiqueter automatiquement les messages entrants, optez pour Étiqueter l'en-tête du message dans la liste déroulante.

  10. Cliquez sur OK pour confirmer les paramètres.
  11. Cliquez sur OK dans la boîte de dialogue Filtres de messages. Le filtre commencera à filtrer les messages entrants dès que vous aurez cliqué sur OK.

Pour gérer les filtres :

  1. Ouvrez une fenêtre Courrier.
  2. Ouvrez le menu &tasksMenu.label; et choisissez &filtersCmd.label;. La boîte de dialogue Filtres de messages s'affiche.
  3. Si vous avez plusieurs comptes de messagerie, sélectionnez celui dont vous désirez gérer les filtres.
  4. Sélectionnez parmi les options suivantes :
    Activer ou désactiver un filtre :
    Cochez la case à droite du nom du filtre pour activer le filtre et décochez-la pour le désactiver.
    Modifier un filtre :
    Sélectionnez le filtre et cliquez sur Édition (ou double-cliquez sur le nom du filtre). La boîte de dialogue Modifier le filtre s'ouvre pour vous permettre d'effectuer vos changements.
    Supprimer un filtre :
    Sélectionnez le nom du filtre et cliquez sur Effacer.
    Pour modifier l'ordre d'application des filtres :
    Dans la liste, cliquez sur le nom d'un filtre, puis sur Vers le haut ou Vers le bas pour le déplacer dans l'ordre d'application.

    Remarque : les filtres sont appliqués à chaque message entrant dans l'ordre de votre choix, jusqu'à ce qu'une action de filtre entraîne la suppression du message ou son déplacement dans un autre dossier.

  5. Cliquez sur OK lorsque vous avez terminé de gérer les filtres. Si vous avez créé un nouveau filtre, il commencera à filtrer les messages entrants dès que vous aurez cliqué sur OK.

Remarque : si vous supprimez un dossier utilisé pour stocker vos messages filtrés, le filtre ne fonctionnera plus. Les messages entrants correspondant aux critères de filtrage restent alors dans le dossier Courrier entrant. Si vous renommez ou déplacez un dossier utilisé pour stocker des messages filtrés, le filtre sera automatiquement mis à jour pour utiliser le nouveau dossier.

Astuce : les filtres ne s'appliquent pas aux messages existants. Si vous avez des messages que vous souhaitez déplacer vers un autre dossier, vous pouvez effectuer une recherche de ces messages puis les archiver dans le dossier en question.

&brandShortName; Courrier & Groupes permet de rechercher rapidement du texte dans un message, de chercher des messages par objet, expéditeur ou d'utiliser une combinaison de critères pour soumettre tous les messages à une recherche approfondie dans un dossier, groupe ou compte de messagerie spécifique.

Pour rechercher du texte dans un seul message :

  1. Ouvrez une fenêtre Courrier.
  2. Sélectionnez le message, ouvrez le menu &editMenu.label; et choisissez &findCmd.label;.
  3. Entrez le texte à rechercher.
  4. Cliquez sur Rechercher pour trouver la première occurrence.
  5. Continuez à cliquer sur Rechercher pour trouver d'autres occurrences ou cliquez sur &undoDefaultCmd.label; une fois votre recherche terminée.
  6. Si vous avez fermé la fenêtre Recherche dans la page, sélectionnez &findAgainCmd.label; dans le menu &editMenu.label; pour continuer à rechercher le texte dans le reste du message.

Pour rechercher rapidement des messages par objet ou par expéditeur :

  1. Ouvrez une fenêtre Courrier.
  2. Dans le champ &searchSubjectAndFrom.emptytext;, entrez le texte du sujet ou le nom de l'expéditeur que vous désirez trouver. Vous pouvez entrer une partie du sujet ou de l'expéditeur ou entrer le nom exact que vous désirez trouver.

    Dès que vous arrêtez d'écrire, &brandShortName; Courrier & Groupes affiche les messages qui possèdent une occurrence ou un expéditeur contenant le texte recherché que vous êtes en train d'entrer.

  3. Une fois votre recherche terminée, cliquez sur Effacer pour supprimer le texte de recherche et afficher tous les messages.

Recherche précise de messages

Pour rechercher des messages dans des dossiers de messagerie ou de groupes :

  1. Ouvrez une fenêtre Courrier.
  2. Ouvrez le menu &tasksMenu.label; et choisissez &searchMailCmd.label;. La boîte de dialogue Rechercher les messages apparaît.
  3. Dans la liste déroulante Rechercher des messages dans, choisissez le compte, le groupe ou le dossier dans lequel vous souhaitez effectuer une recherche.
  4. Cochez Rechercher dans les sous-dossiers pour inclure tous les sous-dossiers dans la recherche.
  5. Sélectionnez Rechercher sur le système local pour rechercher uniquement les messages des groupes de discussion ou des comptes IMAP qui ont été enregistrés localement.

    Note : La case à cocher sera désactivée s'il n'est pas possible de faire une recherche sur un dossier de messages distant.

  6. Sélectionnez l'option correspondante pour rechercher des messages soit en respectant toutes les conditions soit d'après au moins une des conditions que vous choisirez.
  7. Utilisez la liste déroulante pour indiquer les critères de recherche (par exemple Objet et contient) puis entrez le texte ou la phrase que vous recherchez.

    Astuce : pour chercher des messages contenant un en-tête non listé dans la première liste déroulante (par exemple, si vous recherchez des messages qui incluent un champ "Resent-From"), choisissez Personnaliser et entrez l'en-tête dans lequel vous voulez chercher. &brandShortName; Courrier & Groupes ajoutera le nouvel en-tête à la liste déroulante. Ainsi, vous pourrez le choisir pour d'autres règles de messages. Assurez-vous d'avoir correctement saisi le nom du champ, car &brandShortName; Courrier & Groupes ne peut trouver que des correspondances exactes.

  8. Cliquez sur Plus pour ajouter des critères et sur Moins pour enlever le dernier critère de la liste.
  9. Cliquez sur Rechercher pour commencer la recherche ou sur Effacer pour réinitialiser les critères. Les résultats de la recherche apparaissent dans la zone Résultats de la boîte de dialogue Rechercher les messages.
    • Pour ouvrir un message et le lire, sélectionnez-le et cliquez sur Ouvrir ou double-cliquez dessus.
    • Pour trier les messages dans un ordre différent, cliquez sur la colonne à partir de laquelle trier.
    • Pour déplacer ou copier un message de la zone Résultats dans un autre dossier, sélectionnez-le et choisissez le dossier de destination dans la liste déroulante Classer. Si ce dossier est dans le même compte, le message y est déplacé. Si le dossier de destination appartient à un autre compte, il y est dupliqué.
    • Pour supprimer un message de la zone Résultats, sélectionnez le message puis cliquez sur Effacer.
    • Pour ouvrir le dossier dans lequel se trouve un message, sélectionnez-le et cliquez sur Aller au dossier du message.

Utiliser le contrôle des messages indésirables

La fonctionnalité de contrôle des messages indésirables de &brandShortName; peut évaluer vos messages entrants, et repérer les possibles messages indésirables (ou non sollicités). La fonctionnalité utilise une classification bayésienne. Vous commencez par entrainer &brandShortName; en lui indiquant un ensemble de messages qui sont indésirables, et un ensemble de messages qui ne le sont pas. Ensuite, vous le laissez trier lui-même les nouveaux messages. Si &brandShortName; fait une erreur, vous pouvez la corriger.

Pour utiliser le contrôle des messages indésirables :

  1. Commencez par entraîner &brandShortName; à reconnaître les messages indésirables des autres messages. Il y a trois manières de changer le statut de désirabilité de(s) message(s) sélectionné(s) :
    • Ouvrez le menu &msgsMenu.label;, sélectionnez Marquer, et choisissez Comme indésirable ou Comme acceptable.
    • Cliquez sur le bouton Indésirable de la barre d'outils.

    • Cliquez pour modifier la colonne Statut indésirable dans la liste des messages. (Si vous ne le voyez pas, cliquez le bouton le plus à droite () dans la barre d'en-tête de la liste et sélectionnez Statut indésirable depuis la fenêtre surgissante.)

    Quand vous modifier le statut indésirable, une icône de poubelle apparaitra ou non dans la colonne correspondante, pour indiquer le statut indésirable du message sélectionné.

  2. Ouvrez le menu &editMenu.label;, et choisissez &accountManagerCmd.label;. Vous voyez la fenêtre &accountManagerTitle.label;.
  3. Cliquez sur la catégorie &junkSettings.label;, de votre compte courrier.
  4. Activez la fonctionnalité, et &brandShortName; classera automatiquement les messages arrivant (voir les options de contrôle des courriers indésirables. Des détails sur les autres paramètres peuvent être trouvée dans la description du panneau de préférences Paramètres pour les indésirables).
  5. Si vous l'avez entraîné sur des messages contenant des virus, pensez à désactiver la liste blanche (beaucoup de virus envoient des messages pour infecter les personnes contenues dans le carnet d'adresse des machines attaquées).
  6. Assurez vous de corriger les paramètres des courriers indésirables quand l'application fait une erreur de reconnaissance.
  7. Pour analyser les messages existants, sélectionnez les messages à analyser, ouvrez le menu Outils, et choisissez &runJunkControls.label;.

Note : &brandShortName; n'effectuera un contrôle des messages indésirables que lorsque la base de donnée d'entraînement a des informations sur les messages non indésirables. Si le contrôle ne fonctionne pas, sélectionnez quelques messages, et marquez les explicitement comme non indésirables.

Marquer un message comme indésirable

Pour marquer un message comme indésirable, suivez les étapes suivantes :

  1. Sélectionnez un message que vous voulez marquer comme indésirable.
  2. Puis exécutez l'une des actions suivantes :
    • Ouvrez le menu &msgsMenu.label;, sélectionnez Marquer puis choisissez Comme indésirable.
    • Cliquez sur le bouton Indésirable de la barre d'outils. Remarquez que ce bouton change de dénomination une fois cliqué pour devenir Acceptable.
    • Dans la colonne Statut indésirable, cliquez sur le point pour marquer le message. L'icône apparaît dans cette colonne en face du message marqué comme indésirable
  3. Ouvrez le menu &tasksMenu.label; et choisissez Gestionnaire des indésirables…. Activez le gestionnaire des indésirables pour votre compte de messagerie, et &brandShortName; classera automatiquement les messages entrants. Pour plus d'informations, voir la section Gestionnaire des indésirables.

Si vous avez entraîné le gestionnaire des indésirables sur des messages infectés par des vers ou des virus, il est fortement conseillé de désactiver l'option Ne pas marquer les messages comme indésirables si l'expéditeur est dans mon carnet d'adresses.
  5. Au cas où le gestionnaire des indésirables commettrait une erreur dans la classification d'un courriel, corrigez-la manuellement, soit en utilisant l'icône Indésirable/Acceptable, soit en utilisant le menu &msgsMenu.label;.

Détection de l'hameçonnage

L'hameçonnage (phishing en anglais) est une technique courante d'escroquerie, où quelqu'un créé un faux site, destiné à piéger ses destinataires, et à ce qu'ils lui transmettent des informations personnelles comme des numéros de carte de crédit, des nom d'utilisateur et mots de passe, ou encore des numéros de sécurité sociale. En détournant les noms de grandes banques, de vendeurs en ligne et de sociétés de carte de crédit, les hameçonneurs convainquent souvent les destinataires de donner suite à leurs avances.

Dans de nombreux cas, vous recevrez un lien vers une page d'hameçonnage, par un courrier qui se réclame comme émanant d'une origine officielle. Il peut arriver que vous tombiez sur ce type de pages en suivant des liens sur internet, ou par messagerie instantannée.

Astuce : Étant donné qu'une URL forgée peut être très ressemblante à une officielle, il est plus sûr d'utiliser un marque-page que vous avez créé, ou de taper l'URL dans la barre d'adresse, à la main, au lieu de suivre un lien dans un courrier. Pensez toujours au risque d'une URL forgée s'il vous est demandé de vous authentifié ou de fournir des informations personnelles sur un site web.

Le détecteur de tentatives d'hameçonnage de &brandShortName; est activé par défaut. Quand il rencontre un courrier qui semble être frauduleux, une barre d'avertissement s'affichera dans la fenêtre du message.

Si vous pensez que ce courrier n'est pas frauduleux, cliquez sur le bouton &removePhishingBarButton.label; button, et la barre d'avertissement disparaitra.

Quand un utilisateur clique sur un lien dans un courrier qui semble être une URL d'hameçonnage, &brandShortName; affichera une boîte de dialogue à l'utilisateur avant que le site internet ne soit ouvert.

Cette fenêtre apparaitra si l'une des trois conditions suivantes est remplie : le nom de domaine de l'URL est une addresse IP, ou le texte du lien est une URL, dont le nom de domaine ne correspond pas à celui de l'URL réelle.

Note : La détection de l'hameçonnage a une priorité supérieure à celle de la détection des courriers indésirables.

Pour plus d'informations techniques sur le sujet, voyez le document Know your Enemy : Phishing.

À propos du Internet Message Access Protocol (IMAP)

Avantages : vos messages et tout changement les concernant restent sur le serveur, préservant l'espace disque de votre ordinateur. Ce serveur, qui donne accès à une boîte aux lettres toujours à jour, permet de relever vos messages depuis des endroits différents (travail, domicile, etc). La durée de la connexion est plus courte car on ne télécharge initialement que les en-têtes des messages.

Inconvénient : IMAP n'est pas supporté par tous les FAI.

[ Retour au début de la section ]

À propos du Post Office Protocol (POP)

Avantages : tous vos messages sont téléchargés entièrement sur votre ordinateur d'un seul coup. Vous pouvez cependant préciser au serveur de conserver une copie ou d'effacer les messages uniquement lorsqu'ils le sont aussi localement. POP est actuellement supporté par la plupart des FAI.

Inconvénients : si vous utilisez plusieurs ordinateurs, des messages peuvent se trouver sur l'un ou sur l'autre, mais pas sur les deux. POP ne fonctionne pas aussi bien que IMAP avec une connexion à bas débit. Vous ne pourrez pas non plus accéder à tous les dossiers de messagerie à partir de divers endroits.

Veuillez noter que les serveurs POP plus récents ont des fonctionnalités permettant de relever seulement les en-têtes plutôt que les messages complets, comme le permet IMAP. Utiliser cette fonctionnalité permet à POP d'être presque aussi rapide qu'IMAP.

  • Le certificat personnel que vous voulez utiliser pour signer les messages que vous envoyez.
  • Le certificat que vous désirez que vos correspondants utilisent pour chiffrer les messages qu'ils vous envoient.

L'utilisation d'un ou de deux certificats dépend de la politique de votre autorité de certification (AC) qui valide votre certificat. Vous pouvez utiliser un seul certificat pour les deux objets ou deux certificats différents. Même si vous n'en avez qu'un, vous devez le spécifiez deux fois, une fois pour la signature numérique et une fois pour le chiffrement.

Les certificats sélectionné ici seront inclus dans tous les messages signés que vous enverrez. Ces certificats permettront à vos destinataires de vérifier votre signature numérique et de chiffrer les messages qu'ils vous enverront.

Signature numérique

Vous pouvez utiliser le panneau Sécurité pour spécifier la manière de signer vos messages :

&signMessage.label;
Cochez cette case si vous souhaitez signer numériquement tous les messages que vous envoyez (un certificat personnel doit être spécifié avant de pouvoir cocher cette case.).
&signingCert.message;
Si le champ est vide ou s'il affiche un mauvais certificat, cliquez sur Sélectionner pour en choisir un autre.

En ne tenant pas compte de la case à cocher &signMessage.label;, vous pouvez changer d'avis avant d'envoyer un message.

Pour changer le paramètre de signature numérique pour un message que vous écrivez dans la fenêtre de rédaction, cliquez sur la petite flèche sous l'icône &securityMenu.label; dans la barre d'outils et sélectionnez ou désélectionnez &menu_securitySign.label;. Pour plus de détails, voir Signer numériquement et chiffrer un nouveau message.

Chiffrement

Vous pouvez utiliser le panneau Sécurité pour spécifier comment utiliser habituellement le chiffrement lors de l'envoi de messages :

&neverEncrypt.label;
Sélectionnez cette option si vous ne voulez jamais utiliser le chiffrement, ou seulement occasionnellement.
&alwaysEncryptMessage.label;
C'est l'option à sélectionner lorsque l'on souhaite utiliser systématiquement le chiffrement. Si vous n'avez pas tous les certificats nécessaires, le messages ne sera pas envoyé à moins que vous ne désactiviez le chiffrement pour ce message en particulier.
&encryptionCert.message;
Si le champ est vide ou s'il affiche un mauvais certificat, cliquez sur &encryption.certificate.button; pour en choisir un autre.

En ne tenant pas compte de l'option de chiffrement choisie, vous pouvez changer d'avis avant d'envoyer un message.

Pour changer le paramètre de signature numérique pour un message que vous écrivez dans la fenêtre de rédaction, cliquez sur la petite flèche sous l'icône &securityMenu.label; dans la barre d'outils et sélectionnez ou désélectionnez &menu_securitySign.label;. Pour plus de détails, voir Signer numériquement et chiffrer un nouveau message.

Préférences Courriers et groupes - &pref.character.encoding.title;

Les préférences de l'encodage des caractères vous permettent de choisir comment les messages sont encodés quand ils sont affichés ou créés, dans tous les comptes. Si vous n'êtes pas déjà en train de voir les paramètres de l'encodage des caractères, suivez ces étapes :

  1. Ouvrez le menu &brandShortName; &editMenu.label; et choisissez &preferencesCmd.label;. Vous voyez la boîte de dialogue des &prefWindow.title;.
  2. Dans la catégorie Courriers et groupes, cliquez &pref.character.encoding.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur Courriers et groupes pour déplier la liste.
&viewDefaultCharset.label;
Cliquez sur cette liste déroulante pour choisir l'encodage que vous voulez que Courrier et groupes utilise comme encodage par défaut pour les courriers et messages de groupes arrivant. Ceci est recommandé si vous recevez généralement des messages dont l'encodage (charset MIME) n'est pas spécifié, par exemple si vous lisez des groupes internationaux.
&replyInDefaultCharset2.label;
Sélectionnez ceci pour appliquer l'encodage par défaut à tous les messages.

Astuce : Vous pouvez, plus tard, voir ou changer l'encodage pour un dossier en particulier. Dans la fenêtre Courrier, choisissez un dossier dans la liste des dossiers de courriers. Ouvrez le menu Affichage, et choisir Encodage des caractères.

&useMIME.label;
Choisi d'utiliser pour Courriers et groupes l'encodage MIME Quoted-Printable lors de l'envoi de messages qui utilisent des caractères encodés sur 8 bits (par exemple, Latin ISO-8859-3).
&sendDefaultCharset.label;
Sélectionnez l'encodage des caractères que vous voulez que Courriers et groupes utilise par défaut pour les courriers et messages de groupes sortant. Notez que cet encodage n'est pas utilisé pour répondre à un message. Dans ce cas, c'est l'encodage des caractères du message auquel vous répondez qui sera utilisé par défaut. Choisissez Toujours utiliser cet encodage de caractères par défaut dans les réponses pour utiliser l'encodage par défaut pour les messages sortant, même lorsque vous répondez.
