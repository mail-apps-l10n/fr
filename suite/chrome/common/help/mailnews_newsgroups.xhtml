S'abonner à des groupes

Si vous avez configuré un compte sur un serveur de groupes, vous pouvez vous abonner à des groupes (également appelés groupes de discussion).

Pour vous abonner à un groupe :

  1. Ouvrez une fenêtre Courrier.
  2. Ouvrez le menu &fileMenu.label; et choisissez &subscribeCmd.label;. La boîte de dialogue Abonnement s'affiche.
  3. Au besoin, cliquez sur la liste déroulante Serveur pour faire le choix d'un serveur.
  4. Cochez les cases à droite des groupes auxquels vous désirez vous abonner.
  5. Cliquez sur le bouton S'abonner. Une marque s'affiche à côté de chaque groupe auquel vous voulez vous abonnez. Cliquez sur &unsubscribeNewsgroupCmd.label; pour annuler un abonnement.
  6. Cliquez sur OK. La liste des groupes auxquels vous êtes abonné s'affiche dans la fenêtre Courrier.

Si vous utilisez un serveur de messagerie IMAP, vous pouvez également vous abonner à des dossiers de messagerie situés sur un serveur IMAP. Votre dossier Courrier entrant est un type de dossier de messagerie. Respectez les instructions qui précèdent pour vous abonner, mais sélectionnez un serveur de messagerie IMAP dans la liste déroulante Serveur. Pour plus d'informations, voir la section Partage de dossier avec d'autres utilisateurs (IMAP seulement).

Lire des messages de groupes

Lorsque vous ouvrez votre serveur de groupes, la liste des groupes auxquels vous êtes abonné s'affiche. &brandShortName; télécharge les en-têtes des nouveaux messages de chaque groupe.

Pour lire les messages des groupes :

  1. Ouvrez une fenêtre Courrier.
  2. Double-cliquez sur l'icône d'un serveur pour afficher ses groupes. Si la liste est vide, c'est que vous n'êtes abonné à aucun groupe.
  3. Cliquez sur le nom d'un groupe pour voir ses messages.
  4. Cliquez sur un message pour le lire. Cliquez sur l'icône Discussion pour afficher toutes les réponses au message d'origine. Vous pouvez cliquer sur un en-tête pour afficher le message correspondant. Vous pouvez lancer une nouvelle discussion ou répondre à un message d'une discussion existante.

Envoyer un message sur un groupe

Pour lancer un nouveau fil de discussion :

  1. Dans la fenêtre Courrier, sélectionnez un groupe dans la liste des groupes auxquels vous êtes abonné.
  2. Cliquez sur l'icône Rédiger.
  3. Rédigez le message et cliquez sur Envoyer pour le poster.
  4. Cliquez sur Relever pour voir votre message sur le groupe.

Contribuer à des discussions en cours

Pour envoyer une réponse au groupe :

  1. Dans la liste des messages, sélectionnez celui auquel vous souhaitez répondre.
  2. Cliquez sur &replyMsgCmd.label;.
  3. Rédigez le message et cliquez sur Envoyer pour le poster.

Pour répondre à une personne ainsi qu'à l'ensemble du groupe :

  1. Dans la liste des messages, sélectionnez celui auquel vous souhaitez répondre.
  2. Cliquez sur &replyToAllMsgCmd.label;.
  3. Composez le message et cliquez sur Envoyer pour le poster.

Pour faire suivre les réponses d'un message sur un autre groupe :

  • Cliquez sur &replyMsgCmd.label; et sélectionnez Faire suivre à dans la liste déroulante des types de destinataires. Les réponses ultérieures seront envoyées au groupe spécifié.

Surveiller un fil de discussion

Pour surveiller les messages non lus dans les fils qui vous intéressent :

  1. Sélectionnez un message dans une discussion.
  2. Ouvrez le menu &msgsMenu.label; et choisissez Surveiller la discussion. Une icône s'affiche à gauche de la ligne du message.
  3. Pour surveiller d'autres fils, répétez les étapes 1 et 2 pour les messages d'autres fils.
  4. Lorsque vous êtes prêt à observer les messages de ces fils, ouvrez le menu &viewMenu.label;, sélectionnez &threads.label;, puis &watchedThreadsWithUnreadCmd.label;. &brandShortName; Courrier & Groupes n'affichera que les fils surveillés contenant des messages non lus.
  5. Pour réafficher tous les messages du groupe, ouvrez le menu &viewMenu.label;, choisissez &selectThreadCmd.label;, puis Tous.

Pour ignorer un fil de discussion :

  1. Sélectionnez un message dans un fil de discussion.
  2. Ouvrez le menu &msgsMenu.label; et choisissez Ignorer la discussion. &brandShortName; Courrier & Groupes marque tous les messages du fil comme lus, ainsi que les nouvelles réponses correspondant à ce fil de discussion.
  3. Pour afficher les fils ignorés, dans le menu Afficher, choisissez Messages, puis &ignoredThreadsCmd.label;.

Supprimer un groupe

Pour supprimer un groupe de la liste, vous pouvez effectuer l'une des actions suivantes :

  • Cliquez avec le bouton droit de la souris sur le groupe correspondant et sélectionner &folderContextUnsubscribe.label; dans le menu contextuel qui apparaît.
  • Sélectionnez le groupe que vous désirez supprimer puis appuyez sur la touche Supprimer.
