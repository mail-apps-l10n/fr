Préférences d'apparence

Cette section décrit comment utiliser le panneau principal des préférences d'apparence. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Sélectionnez &pref.appearance.title;. Double-cliquez sur la catégorie &pref.appearance.title; pour développer la liste, puis choisissez les préférences que vous voulez éditer.

Dans cette section :

  • &pref.appearance.title;
  • &pref.content.title;
  • &pref.fonts.title;
  • &pref.colors.title;

Préférences d'apparence - &pref.appearance.title;

Le panneau des préférences de l'apparence permet de définir les options de démarrage de &brandShortName; et de personnaliser l'interface utilisateur :

&onStartLegend.label; :
Sélectionnez les composants que vous voulez utiliser au démarrage de &brandShortName;.
&showToolsLegend.label; :
&picsNtextRadio.label; :
Sélectionnez cette option si vous désirez que les icônes des boutons des barres d'outils soient sous-titrées.
&picsOnlyRadio.label; :
Sélectionnez cette option si vous ne voulez voir que les icônes des boutons des barres d'outils.
&textonlyRadio.label; :
Sélectionnez cette option si vous ne voulez voir que le texte des boutons des barres d'outils.
&showHideTooltips.label; :
Cochez cette case pour voir apparaître des infobulles quand le curseur est au-dessus d'un élément de l'interface utilisateur de &brandShortName; et de certaines pages Web.
&pref.locales.title; :
Ce paramètre vous permet de changer la langue utilisée dans l'interface utilisateur de &brandShortName;. Des langues supplémentaires peuvent être installées à partir de la page de démarrage de &brandShortName;.

Remarque : Vous devez redémarrer &brandShortName; pour qu'un changement de languesoit effectif.

Préférences d'apparence - &pref.content.title;

Cette section décrit comment utiliser le panneau des préférences &pref.content.title;. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName; &editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Dans la catégorie &pref.appearance.title;, cliquez sur &pref.content.title;. (Si aucune catégorie n'est visible, double-cliquez sur &pref.appearance.title; pour développer la liste).

Le panneau de préférences &pref.content.title; vous permet de changer les paramètres qui influencent la façon dont le contenu des sites Web et des messages apparaissent dans &brandShortName;.

&siteIcons.label; :
&useSiteIcons.label;
Cochez cette case si vous voulez voir les icônes spécifiques aux sites, si disponibles, à la place de l'icône des marque-pages  :
Icône de marque-pages (apparaît à gauche de l'adresse).
Les icônes de sites Web sont affichées à gauche de la barre d'adresse et des onglets du Navigateur.
&useFavIcons.label;
Si la page elle-même ne définit pas d'icône de site Web, activer ce paramètre fera chercher à &brandShortName; une favicône sur le serveur pour l'utiliser à la place.
&useSmoothScroll.label; :
Cette option vous permettra de faire défiler une page Web sans à-coup.
&allowHWAccel.label;
&enableAutomaticImageResizing.label; :
Lorsqu'une image est plus grande que la taille de votre écran, cette option permet à &brandShortName; de la redimensionner pour qu'elle s'adapte au mieux à la taille de la fenêtre du navigateur.
&textZoomOnly.label;
Cochez cette case si vous voulez que &brandShortName; ne redimensionne que la taille du texte des sites Web en utilisant la fonction Zoom. Si cette case n'est pas cochée, toute la page, y compris les images, sera redimensionnée.
&blockAutoRefresh.label;

Remarque : Consultez les différents panneaux pour les couleurs, les polices et les langues pour personnaliser encore plus l'apparence du contenu et la section &pref.security.title; pour les paramètres relatifs à la confidentialité et à la sécurité qui pourraient également influencer la façon dont le contenu est affiché.

Préférences d'apparence - &pref.fonts.title;

Cette section décrit comment utiliser le panneau des préférences des polices de caractères. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Sous la catégorie &pref.appearance.title;, sélectionnez &pref.fonts.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &pref.appearance.title; pour développer la liste.

Le panneau des préférences des polices permet de configurer le type et la taille des polices d'une page.

Remarque : Il se peut que certaines polices ne soient pas sélectionnables parce que la langue en cours n'a pas de polices disponibles pour ce style.

&language.label; 
Choisissez un groupe de langue. Par exemple, pour configurer l'encodage par défaut des polices pour les langues occidentales, choisissez le groupe &font.langGroup.latin1;. Pour utiliser un encodage absent de la liste, sélectionnez &font.langGroup.user-def; For more information, including User Defined, see Selecting Character Encodings and Fonts. All settings below, except for the checkbox, are stored per language group; each can have its own set of font definitions..
&proportional.label; 
Choisissez si le texte proportionnel doit être avec serif (tel que Times New Roman) ou sans serif (tel que Arial). Il est aussi possible de spécifier la taille de la police que vous voulez utiliser pour le texte proportionnel. Le texte proportionnel est variable en largeur, donc les caractères et les lettres varient en largeur.
&serif.label; 
Sélectionnez la police serif que vous désirez utiliser pour les pages Web.
&sans-serif.label; 
Sélectionnez la police sans serif que vous désirez utiliser pour les pages Web.
&cursive.label; 
Sélectionnez la police cursive que vous désirez utiliser pour les pages Web.
&fantasy.label; 
Sélectionnez la police fantaisie que vous désirez utiliser pour les pages Web.
&monospace.label; 
Sélectionnez la police monospace (telle que Courier) que vous désirez utiliser pour les pages Web. Le texte monospace est fixe en largeur, chaque caractère ou lettre occupe le même espace.
&minSize.label; 
Sélectionnez la plus petite taille de police à utiliser dans les pages Web.
&useDocumentFonts.label; 
Cochez cette case pour garder la taille et la police proposées par la page Web, plutôt que celles de vos préférences.

Préférences d'apparence - &pref.colors.title;

Cette section décrit comment utiliser le panneau des préférences de couleurs. Si vous n'êtes pas déjà en train de le visualiser, suivez ces étapes :

  1. Ouvrez le menu &brandShortName;&editMenu.label; et sélectionnez &preferencesCmd.label;.
  2. Sous la catégorie &pref.appearance.title;, sélectionnez &pref.colors.title;. Si aucune sous-catégorie n'est visible, double-cliquez sur la catégorie &pref.appearance.title; pour développer la liste.

Le panneau des préférences de couleurs vous permet de définir les couleurs de fond et de texte des pages Web :

&color; :
Cliquez sur les blocs colorés pour choisir les couleurs du texte et du fond des pages Web.
&useSystemColors.label; :
Sélectionnez cette option pour utiliser les réglages de couleurs de votre système.
&links; :
Cliquez sur les blocs colorés pour choisir la couleur des liens visités et non visités des pages Web.
&underlineLinks.label; :
Sélectionnez cette option pour souligner les liens dans les pages Web.
&someProvColors; 
&alwaysUseDocumentColors.label; :
Autorise les pages Web à choisir leurs couleurs et leur fond d'écran.
&useMyColors.label; :
Permet de choisir les couleurs et le fond d'écran.
